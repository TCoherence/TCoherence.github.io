Rakuten 返现日报｜2026-05-07

一句话结论：Big Give Week 进入 Day 4（5/4–5/11），主页与 30+ 商家页仍维持 15% 高位（Macy’s 例外，已回落 2%；Saks 已结束 15x 短窗），Incogni / NordVPN 100%、SoFi Banking $300、Kraken $150–$160 三档新客闪购仍在档；今日入手优先级：母亲节美妆礼（Ulta 15%、Dermstore 20x）、家居宠物（PetSmart / Petco / Sonos / KitchenAid 15%）、暑期 OTA（Expedia Hotels 6% + Things to Do 15%、Viator 15%、Tripadvisor 15%）。

摘要

  • Big Give Week Day 4：rakuten.com/f/big-give-week 落地页确认 400+ 商家 15% 池。可叠加 Amex MR / Bilt 1:1 兑换（Bilt 5/15 一次性 1:1 推广窗仍在跑），实际效率最高的是 Bilt 出口 15× MR/Bilt 点数线。
  • 主体仍稳：Ulta 15% / PetSmart 15% / Petco 15% / DSW 15% / LEGO ≤15% / Sonos 15% / Adidas ≤15% / Gap 15% / Bombas 15% / Groupon 15% / TEMU 15% / AliExpress 15% / Office Depot ≤15% / VistaPrint 15% / CVS 15%（线上）。
  • 已下沉案例（不要盲点旧文）：Macy’s 已回落 2%（Day 1–2 的 15% 已结束）；Saks 也回落 2%（Frequent Miler 报道的 15x 仅 5/4 当天短窗）；Lenovo 仅 2%（虽列在 Big Give 名单，但商家页未上调）。
  • 新客闪购天花板继续：Incogni 100% / NordVPN 100% 新客首单；SoFi Banking $300（was $125）、SoFi Invest $100（was $50）、Kraken $150–$160（首笔 $200 交易）三档现金返还仍在跑。
  • 风险与窗口：Big Give Week 周日（5/11）结束，多数商家会在 Day 5–6 起波段回落（参考 Macy’s / Saks / Lenovo 已回落），优先在 Day 4–5 决策；Travel 类 “up to 15%” 实为分品类（Expedia Hotels 6% / Things to Do 15% / Packages 2%），下单前看产品页。

今日高返现商家

  • Incogni 100% 新客闪购 — 100%（was 20%），新客首期订阅，限 1 次/会员，40 天验证期，refund/gift card 不计；数据删除服务首单等于回本。(Rakuten Incogni 商家页, DoC 100x 总结)
  • NordVPN 100% 新客闪购 — 100%（was 20%），仅新客首签且只对 NordVPN Basic 计算（NordPass / NordLocker / NordProtect / 专享 IP / B2B 不算）。(Rakuten NordVPN 商家页)
  • Ulta Beauty 15%（was 2%） — 母亲节美妆首选；Dior、Rewards / Rewards Discount 抵扣部分、礼卡不计；只用 Rakuten 列出的码。(Rakuten Ulta 商家页)
  • PetSmart 15%（was 2%） — 1 单/Trip；首单 Autoship 后续不计；礼卡 / Vet / Adoption / Pharmacy / 数量>100 / Treats Rewards 抵扣部分均不计。(Rakuten PetSmart 商家页)
  • Petco 15%（was 2%） — 仅线上单计 15%，门店 0%；与 PetSmart 二选一比促销码。(Rakuten Petco 商家页)
  • DSW 15%（was 2%） — Gucci / Bottega / Ferragamo / Off-White / Versace 等奢牌不计；母亲节鞋履与暑期出行鞋款合适。(Rakuten DSW 商家页)
  • LEGO 上至 15%（was 2.5%） — Exclusives & hard-to-find 仅 3%，需开启第三方 Cookie；常规 set + Mother’s Day 礼合算。(Rakuten LEGO 商家页)
  • Sonos 15%（was 2%） — 全线 15%；可叠 Sonos 自家促销 + 信用卡音视频码。(Rakuten Sonos 商家页)
  • Bombas 15%（was 2%） — 全场 15%，礼袜 / 母亲节家庭装合适；只用 Rakuten 上列码。(Rakuten Bombas 商家页)
  • Gap 15%（was 2%） — 历史第一档；可与 Gap Inc. 自家促销叠（注意未列码会失效）。(Rakuten Gap 商家页)
  • TEMU 15%（was 3%） — App 单不计、必须美国地址、Cash App Pay 不计、车里有货时点击不计；下单前清空购物车再点 Rakuten。(Rakuten TEMU 商家页)
  • AliExpress 15%（was 2%） — 批发 / Reseller / 礼卡不计；线上次日入账。(Rakuten AliExpress 商家页)
  • Groupon 15%（was 2%） — Costco Membership 单只 4%；Groupon+、Groupon Bucks 抵扣部分不计；本地服务 / 旅行 deal 性价比拉满。(Rakuten Groupon 商家页)
  • Office Depot 上至 15%（was 2%） — Accessories 走 15%；电脑 / 电子 / 家具仅 2%；先看品类再下手。(Rakuten Office Depot 商家页)
  • CVS 15% 线上（was 5%） — 仅线上 15%、门店 2.5%；处方 / CarePass / OTC / COVID 测试 / App 单 / 礼卡 / ExtraBucks 抵扣部分不计；> $200 / 同款 > 8 件可能被判转售。(Rakuten CVS 商家页)

限时闪购返现

  • SoFi Banking $300（was $125） — 新开支票或储蓄账户即得；现金可选 Amex MR / Bilt 出口；本档为 Big Give 期间被推到 $300 的当前最高位。(Rakuten SoFi 商家页)
  • SoFi Invest $100（was $50） — 新开 Active Invest 账户并 45 天内入金 $100 即触发；与 Banking 不冲突。(Rakuten SoFi 商家页)
  • Kraken $150–$160（was $50） — 首笔 $200+ 交易（30 天内）；不能叠 Kraken 端 referral / coupon；DoC 5/3 起拉到 $160 档。(Rakuten Kraken 商家页, DoC Kraken 总结)
  • Expedia Hotels 6% / Things to Do 15% / Packages 2% / Cars 1% / Flights $1.25 — 标 “Up to 15%” 实分品类；夏季 OTA 优先 Things to Do（与 Viator 同档）+ Hotels 6%。(Rakuten Expedia 商家页)
  • Viator 15%（was 2%） — 入账等行程完成后两周；Theme park / virtual / 礼券 / Travel Agent Program / 折扣码 / 学生折扣不计；不要切 App。(Rakuten Viator 商家页)
  • Tripadvisor 15%（was 4%） — 历史高位；Things to Do / Tours 同 Viator 路线，二选一对比码。(Rakuten Tripadvisor 商家页)
  • VistaPrint 15%（was 2%） — 母亲节相册 / 卡片 / 个性化礼合适；Big Give 加码罕见。(Rakuten VistaPrint 商家页)
  • Adidas 上至 15%（was 2%） — Samba / Gazelle / Spezial / EVO SL 仅 4%；其余款 15%。(Rakuten Adidas 商家页)

新上线商家与活动

  • Big Give Week 落地页（5/4–5/11，400+ 店） — 主入口，每天部分名单微调；建议加入 watchlist 每天回看。(rakuten.com/f/big-give-week, Rakuten 官方 Blog)
  • Bilt × Rakuten 1:1 推广窗（含 5/15 出账） — 5/15 这次 Rakuten → Bilt 1:1 出口仍在窗内；本月 Big Give 的 15× 直接转 15× Bilt 点数（Bilt 估值 ≥ 1.5 cpp 时实测远超 cash 回报）。(DoC Rakuten×Bilt 1:1)
  • Rakuten Amex Card $200 SUB（与 Big Give 联动） — Rakuten Amex 卡 SUB 回到 $200，能吃 Rakuten 商城 / Dining 增点；新会员 5/4–5/11 是开卡 + 跑大额最高效窗口。(DoC Rakuten Amex)
  • Rakuten 新会员 $50 / 5,000 点 高位推荐窗 — 历史最高 referral 仍在跑（一般 $30），5/4–5/11 拉新最划算。(DoC Rakuten Referral)

值得关注的叠加策略

  • 母亲节美妆礼线：Ulta 15% × Ulta Beauty Rewards × Amex Offer（Ulta / Dior 当前若有命中）× Bilt 出口 → 15× Bilt（≥ 22.5¢/$1 等价值）。注意 Dior / Rewards 抵扣部分不计 Rakuten。
  • 暑期 OTA 二选一：Viator 15% 或 Tripadvisor 15% → 看具体行程哪边低 + Things to Do 同档；放弃 Expedia Packages 2%。Hotels 走 6% 仅当无更高直订渠道。
  • 家居宠物三件组：Sonos 15%（Sonos 端母亲节促销码）+ PetSmart / Petco 15%（自动续订外的首单）+ LEGO ≤15%（避开 Exclusives 3%）。
  • 新客闪购双开：Incogni 100% + NordVPN 100%（不可用未列码），用 Rakuten 推荐链接拿 $50 / 5,000 高位推荐 后再做，订阅记得 30 / 40 天验证窗内别取消测试单。
  • 银行/Crypto 现金组合：SoFi Banking $300 + Kraken $150–$160（首笔 $200 限价单）= 一周内净 ~$450 现金，比 5/15 后 1:1 Bilt 出口更适合现金需求方。
  • 信用卡交叉：所有 Big Give 商家 → Rakuten Amex SUB $200 跑账 + Card-linked Amex Offer（Big Give 期间 Amex 通常会推 Macy’s / DSW / Adidas / Sonos 加码）。
