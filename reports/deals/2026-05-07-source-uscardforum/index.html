美卡论坛日报｜2026-05-07

一句话结论：5/5–5/7 三股主线收敛——(1) Hyatt 新表 T-13 进入最后冲刺，论坛回帖 5/7 仍密集，"upper 是新常态、lowest 是给你看"已成共识；(2) 5/6 集体诉讼/赔款窗口同日开三个：Apple Intelligence Siri 集体和解（$25/设备、最高 $95）、Fidelity Data Security Settlement（$100 + CA $50 可叠加）、IHG Q2 4 晚 8K / 2 晚 2K 注册期开放（5/19 截止注册、入住 5/20–8/31）；(3) 银行/美卡日常侧新增 US Bank “刷 50 次给 $50”（5/1–6/30、5/31 enroll 截止）、Atmos Summit 50% 机票折扣（travel 9/8–11/18）、CFF + Chase Travel 9x 叠加、C1 Travel Portal price match $200 goodwill 实战、Bilt + Amex AS 付租金 5/6 复测确认 DEAD 已一个月。

摘要

5/7 当天高密度新增 1 条（US Bank 刷 50 次拿 $50）；5/6 是产生最多 hard signal 的一天（Apple Siri / Fidelity / IHG Q2 注册期开启 / Atmos Summit 50% off flights / Oura PayPal 翻车 / IHG Traveler vs Premier / 0/24 求建议 / Bilt+Amex DEAD 复测 / 大延误 6h+ Amex 险讨论）。Carryover 主线仍是 Hyatt 5/20 新表（T-13 到 T-14）、Chase 5/1–5/2 杀全家两连 + 5/5 27 张卡 recon、BoA Unlimited Cash 6/30 不动卡警告、HSBC Q2 $6K 持续 DP、Citi chat 转卡 AA → Strata Elite 通路。本期共 14 条 mainline 高置信（10 条新增 / 4 条 carryover），1 条进 watchlist。

热门讨论与数据点

  • US Bank “刷 50 次给 $50” 5/7 主帖：5/1–6/30 期间消费、enroll 截止 5/31、阶梯 10 笔=$10 / 25 笔=$25 / 50 笔=$50，每笔 ≥ $10；OP 评 “差不多一天一次”。版主已合并入既有 UAR $100/50 笔老帖（uscardforum #504172）。
  • Apple Intelligence Siri 集体和解 5/6 preliminary approval：覆盖 2024/06/10–2025/03/29 购买的 iPhone 15 Pro/Pro Max + 16 全系列；$25/设备、最高 $95，45 天内寄申领通知（uscardforum #503870）。
  • Fidelity Data Security Settlement 5/6 主帖：基础 $100、CA 居民再叠 $50；版主合并入既有 Fidelity class action 帖；楼内提醒 OP 别在截图里露登录信息（uscardforum #504055）。
  • IHG Q2 5/6 注册期开放：4 晚 8K / 2 晚 2K（二选一），每晚现金价 ≥ $30、OTA / 奖励兑换房不算；注册截止 5/19、入住窗 5/20–8/31。同步 IHG One Rewards 主帖楼层（uscardforum #503649）。
  • C1 Travel Portal price match 5/5 实战：OP 找到 Expedia 比 C1 portal 便宜 $650，电话申诉时差价已被 Expedia 修正、客服仍出 $200 travel credit goodwill。楼内有 $50 / $60 案例对比，规则是新行程对所有人公开、不带 coupon / promo / member discount（uscardforum #503742）。
  • Bilt 用 Amex AS 付租金 5/6 复测确认 DEAD：OP DaNiuGe 多次失败、Michael_1 楼复测 “AS in Bilt 一个月前停止接受 Amex”，Reddit 偶有成功 DP 也已无法复现（uscardforum #503837）。
