Frontier Radar — 2026-05-05

综述

本日前沿三大主线并行：

  1. Anthropic 5/5 重磅落子华尔街 —— Opus 4.7 全平台 GA、Microsoft 365 Excel/PowerPoint/Word/Outlook 跨应用上下文 GA、Moody’s 6 亿公司信评 + 风险数据原生嵌入 Claude，叠加 5/4 Blackstone × Goldman Sachs × Hellman & Friedman $1.5B JV，是对 5/1 国防部 7 厂商 Pentagon 招标出局（Anthropic 拒绝 ‘all lawful’ 条款）的快速对冲——从国防转金融 + 企业治理。
  2. OpenAI 走平台分发路径 —— 5/4 GPT-5.5 / GPT-5.4 / Codex 落地 Amazon Bedrock；同日推 Advanced Account Security（phishing-resistant 登录 + Codex 安全姿态）；TPG/Brookfield/Advent/Bain 锚定 ~$10B 二级 SPV，与 Anthropic JV 形成镜像。
  3. 开源 / 挑战阵营加速 —— xAI Grok 4.3 5/3 GA（1M context、原生视频、降价 ~40%）、Mistral Medium 3.5 4/29 GA（128B dense + 修改 MIT）+ Mistral Workflows、DeepSeek V4 Pro/Flash 1M context promo 持续到 5/31。Google DeepMind Deep Research Max（Gemini 3.1 Pro）+ Workspace Intelligence 是平台增强补丁，Meta 5/2 收购 Assured Robot Intelligence 进入具身硬件路径。

OpenAI

  • 5/4 Advanced Account Security 上线：phishing-resistant 登录、缩短会话、登录告警、自动训练排除，Codex 同步覆盖，附 Yubico 安全密钥包。(OpenAI News, Releasebot OpenAI 更新)
  • 5/4 OpenAI × AWS 落地：GPT-5.5 + GPT-5.4 上 Amazon Bedrock（limited preview）；Codex 进 Bedrock 原生 + Bedrock Managed Agents powered by OpenAI 同步亮相；AWS commitments 可计入 Codex 用量。(AWS Weekly Roundup 5/4)
  • 5/4 ~$10B 二级 SPV 落地：TPG 锚 ~$4B + Brookfield/Advent/Bain。CFO Friar 4/29 表态 2026 年内 IPO 无望，二级流通继续承担估值锚定。(Mean.CEO AI News 5月)
  • Musk v. Altman week 1 close：5/1 Musk 自承 xAI 蒸馏 OpenAI；5/4 OpenAI 律师披露 Musk 在和解谈判后向 Altman / Brockman 发威胁短信；mid-May 宣判前关键证词阶段。(MIT Tech Review, TechCrunch)
  • GPT-5.5 收入 + Pentagon 招标：API 收入增速 >2x 历代、Codex 7 天翻倍；5/1 Pentagon 7 厂商榜单包括 OpenAI（与 Google / Microsoft / AWS / NVIDIA / SpaceX / Reflection AI 并列）。(Pentagon AI Contracts 5/1)

Anthropic

  • 5/5 Claude Opus 4.7 GA：Claude 产品/API + AWS Bedrock + Vertex AI + Microsoft Foundry 全平台。视觉解析高分辨率提升、专业任务’更有品味’是官宣定调。(Anthropic Opus 4.7)
  • 5/5 Wall Street 套件：Microsoft 365（Excel/PowerPoint/Word/Outlook）跨 4 应用统一上下文 add-in GA；Moody’s 全平台原生嵌入 Claude，覆盖 6 亿公司信评 + 风险数据；新增 Verisk / Third Bridge / Fiscal AI / D&B / Experian / GLG / Guidepoint / IBISWorld 连接器，加入既有 LSEG / S&P Capital IQ / Morningstar / PitchBook。(Fortune, AI Business)
  • 5/4 $1.5B JV：Anthropic × Blackstone × Goldman Sachs × Hellman & Friedman 联合企业 JV，定位’把 AI 整合到全美企业’，对冲 5/1 国防出局——重心切金融 + 企业治理。(Inc. Anthropic JV)
  • Claude Security Enterprise public beta：代码漏洞扫描 + 修复建议、定时 / 定向扫描、triage 跟踪、导出 + 工作流集成；同期官方重申 Claude 永远 ad-free。(Mean.CEO Anthropic 5月)
  • 5/1 Pentagon 出局：拒绝 ‘all lawful’ 用途条款（潜在域内大规模监控、全自主武器范畴）；7 厂商分类网络合同最终给 OpenAI / Google / Microsoft / AWS / NVIDIA / SpaceX / Reflection AI。(Pentagon AI Contracts)

Google DeepMind

  • Gemini Deep Research + Deep Research Max：基于 Gemini 3.1 Pro，新增 MCP 支持、原生可视化、跨网页 / 自定义源长程调研。Max 主打 quality + 长 horizon，Deep Research 主打低延迟低成本嵌入到 surface UX。(Google Blog, SiliconANGLE)
  • Workspace Intelligence + 文件创建 GA：跨 Gmail / Chat / Calendar / Drive 实时上下文层；Gemini app 全球 GA chat 内创建 PDF / Word / Excel / Docs / Sheets / Slides。(Releasebot Gemini, Mean.CEO Gemini 5月)
  • Gemini Embedding 2 GA：via Gemini API + Vertex AI；与 4/22 Cloud Next 26 公布的 Gemini Enterprise Agent Platform / Workspace Studio / A2A v1.0 形成全栈 stack 对外（vs OpenAI / Anthropic）。(The Next Web, Google Cloud Next Wrap-up)

Meta

  • 5/5 Code World Model Preparedness Report：HuggingFace Daily 第 4 名，arxiv 2605.00932；评估 CWM（代码生成 + 代码推理）frontier 风险，结论’未引入超越当前 AI 生态额外风险’。今日唯一可识别 frontier-lab 署名的责任披露文档。(arxiv 2605.00932, HF papers)
  • 5/2 收购 Assured Robot Intelligence：机器人 AI 模型 startup 整体并入 Meta 具身路线图。Llama 4 Scout（17B active / 16 expert / 10M context / 单 H100）+ Maverick（17B active / 128 expert / H100 host）继续 LlamaCon 2026 节奏。(Mean.CEO AI News 5月, Meta AI Llama 4)

xAI

  • 5/3 Grok 4.3 全量 API 鋪开：context 拓 1M、原生视频输入 day-1、输入价 ~40% 下降；超低定价 + 进阶语音克隆是亮点。SuperGrok ($30/mo) + X Premium+ ($40/mo) 4 月 beta 完成。(SoftTechHub, Apiyi 中文释义, xAI Docs Release Notes)
  • Musk v. Altman week 1：Musk 庭审自承 xAI 蒸馏 OpenAI 模型 / 蓝图，xAI 法务 narrative 受拷问；Grok 5（6T MoE’全球最大公开模型’）官方 X 帐号已推迟到 Q2 2026 窗口外。(MIT Tech Review, Grok 5 节奏)

Mistral

  • Mistral Medium 3.5 4/29 GA：128B dense（不是 MoE）、merge instruction-following + reasoning + coding 三合一、修改 MIT 开权重，4 GPU 自托管可跑；SWE-Bench Verified 77.6% / $7.50 Mtoken 输出。同步推 Vibe remote agents（async coding，cloud session 可下传本地）+ Le Chat Work mode（multi-step + 并行工具调用）。(Mistral Vibe Remote Agents, RevolutionInAI 评测)
  • 5/4-5/5 Mistral Workflows GA：编排引擎，把 AI 系统从实验推向生产业务流，结构化多步 + 内置可观察性 + 模型可换 + 数据隐私控制；与 Le Chat Work mode 形成上下游。(MarketingProfs AI Update)

Qwen

  • 本日无独立 5/5 新品，延续 4/2 Qwen 3.6-Plus（agentic coding + 原生多模态 + 设计稿/截图直生成前端代码）+ 4/16 Qwen3.6-35B-A3B（HF + ModelScope，Apache 2.0）+ Qwen 3.6-Max-Preview 节奏；MoE 路由稀疏激活已成默认架构。(Caixin 4/2 Qwen3.6-Plus, Qwen 3.6-35B-A3B, HF Qwen 3.6-27B)

DeepSeek

  • DeepSeek V4 Preview 4/24 双模型继续吃市场：V4-Pro（1.6T total / 49B active）+ V4-Flash（284B total / 13B active），1M context、Thinking / Non-Thinking 双模、Expert / Instant chat 入口；Promo pricing 5/31 截止（V4 Pro promo 期价格比 post-promo 低 5×），agentic coding open-source SOTA、世界知识仅次 Gemini-3.1-Pro。(DeepSeek API Docs, HF V4-Pro, CNBC 4/24)

Unverified frontier signals (watchlist 待交叉)

  • Anthropic Claude Mythos 5（10T 参数） 是否已 GA：Mean.CEO 4 月汇编与 Geeky Gadgets 周报均有引述，但红色域 mythos-preview 页面仍是 4/20 Mythos Preview / Project Glasswing 文案，Anthropic 5/5 Wall Street blog 主推 Opus 4.7 而未独立提及 Mythos 5。watchlist 等待官方 system card。(红色域 Mythos Preview, Mean.CEO Anthropic 5月)
  • Grok 5（6T MoE） 仍未发布，xAI 官方 X 推迟到 Q2 2026 窗口外，社区讨论级别；Musk 庭审中 xAI 蒸馏 OpenAI 表态后续可能影响发布节奏。(Grok 5 节奏, MindStudio 路线图)
