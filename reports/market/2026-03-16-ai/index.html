AI 日报｜2026-03-16

一句话结论：算力供给与企业级部署仍在加速，但能源与出口管制约束没有消失，2026 年主线是“高增长 + 强约束”的再平衡。

摘要

截至 2026-03-16 的可验证公开信息显示，AI 产业链继续沿“电力需求上行 -> 芯片与晶圆产能兑现 -> 私有化/主权化基础设施扩张 -> 模型能力升级 -> 行业场景落地”推进（IEA, NVIDIA）。

短期看，NVIDIA 与 TSMC 的财务和月度数据仍支持高景气判断；中期看，电力与政策约束决定增长斜率（TSMC, BIS）。

Energy

IEA《Energy and AI》执行摘要指出：2024 年全球数据中心用电约 415TWh（约占全球用电 1.5%），基准情景下 2030 年升至约 945TWh，且需求在地理上高度集中（IEA）。

对本日报的含义是：能源侧已从背景变量转为一线约束变量，尤其北美大规模推理集群所在区域的并网与调峰能力（IEA）。

Chips

NVIDIA 于 2026-02-25 披露 FY2026 Q4（截至 2026-01-25）业绩：季度营收 681 亿美元、数据中心 623 亿美元，FY2026 全年营收 2159 亿美元（NVIDIA, AP）。

TSMC 4Q25 营收新台币 1.046 万亿元，同比 +20.5%；2026 年 1 月与 2 月月营收同比分别 +36.8% 与 +22.2%，显示高景气延续但月度节奏并非线性（TSMC 4Q25, TSMC Monthly）。

Infra

AWS 于 2025-12-02 推出 AWS AI Factories，强调在客户自有数据中心快速部署高性能 AI 基础设施；2026-03-02 的 Weekly Roundup 继续披露 Bedrock 生态更新（AWS AI Factories, AWS Roundup）。

信号是企业基础设施正在从“公有云优先”走向“公有云 + 专有环境/主权部署”并行，采购约束从算力单价扩展到合规与数据边界（AWS AI Factories）。

Model

Anthropic 于 2026-02-05 发布 Claude Opus 4.6，强调 agentic coding、1M token 上下文（beta），并维持 API 定价不变（Anthropic Opus 4.6）。

模型层竞争点继续从“单次问答”迁移到“长任务稳定性 + 工程可用性 + 单位成本”，对企业侧意味着集成价值上升（Anthropic Opus 4.6）。

Application

Anthropic 与 Salesforce 在 2025-10-14 宣布扩展合作，将 Claude 作为 Agentforce 的优先模型之一，覆盖金融、医疗、网络安全等强监管行业，且部分能力已面向 select customers 可用（Anthropic x Salesforce）。

这表明应用层新增量主要来自“可审计、可合规、可接入现有流程”的企业场景，而非消费端一次性爆款功能（Anthropic x Salesforce）。

层间联动影响

BIS 在 2026-01-13 调整对华半导体许可审查口径，同时 2025-05 有 AI Diffusion Rule 撤销与替代框架信号，政策端呈“边放边管”结构（BIS Jan 2026, BIS May 2025）。

联动路径上，政策变化先影响高端芯片可得性和交付节奏，再传导至基础设施部署与模型训练/推理成本，最终作用到应用上线速度与区域分化。
