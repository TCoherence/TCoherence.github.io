AI 日报｜2026-05-01

一句话结论：白宫亲自踩刹车 Anthropic Mythos 70 家扩展、OpenAI 同步收紧 GPT-5.5 Cyber——前沿 AI 一夜进入"超能力国有化"语义；同时 Q1 财报释放美四巨头 2026 AI capex $725B 与 Anthropic 估值逼近 $900B 的双重资本峰值，开源侧 DeepSeek V4 在 Huawei Ascend 950 上完成 Day-0 适配并撬动 BAT 抢芯片。

摘要

本日（2026-05-01，周五）的核心信号集中在三条互相耦合的轴上：(1) 政府层：白宫据 WSJ 反对 Anthropic 把 Mythos 经 Project Glasswing 扩展给约 70 家公司，理由是算力被分流影响 NSA 等政府用户，且同步起草联邦机构使用 Anthropic 的方案；OpenAI 旋即收紧 GPT-5.5 Cyber 的访问范围、对外承认与 Anthropic 同样姿态。(2) 资本层：Microsoft / Alphabet / Amazon / Meta Q1 财报齐超预期但 capex 全线上调（MSFT $190B、GOOG $180–190B、META $125–145B、AMZN ~$200B），合计 2026 AI capex 进入 $650–725B 区间；Anthropic 同日传 $50B 募资 / $900B 估值两周内落地，且 Q1 LLM 收入 31.4% 全球第一首次反超 OpenAI（29%）。(3) 产业链：Cambricon Q1 营收同比 +160%、股价一日涨 14%；TrendForce 确认 Huawei Ascend / Cambricon / Hygon / 摩尔线程对 DeepSeek V4 完成 Day-0 适配，Huawei Ascend 950 订单被 ByteDance / Tencent / Alibaba 同时抢购，国产链首次以"模型即生态"的速度对齐 Nvidia。

中文圈本日两期高质量播客同时落地：晚点聊 163 期对 DeepSeek V4 的工程级拆解，张小珺商业访谈录 139 期由 Yu Su（苏煜，OSU CS 副教授 / 2025 Sloan / NeoCognition CEO）综述 Agent 技术史与 OpenClaw Moment——后者刚于 4/21 完成 $40M 种子。

Frontier Labs / Frontier Model Radar

  • Anthropic —— 4/30 WSJ → Bloomberg / Reuters 路径：白宫直接表态反对 Mythos 扩展给 ~70 家公司（其中含 Apple / Microsoft / Nvidia 已在 Glasswing 内），理由是 NSA 等政府用户的算力会被分流；Mythos Preview 此前已在 73% expert-level CTF 任务、和"32 步模拟企业网络攻击"上首次端到端成功。(Bloomberg / The Next Web / Crypto Briefing / NextGov)
  • Anthropic 估值与收入 —— 4/30 TechCrunch：48 小时内向投资人定额，目标 ~$50B 募资 / ~$900B 估值，最终估值或更高；The Register / Counterpoint：Q1 2026 全球 LLM 收入份额 31.4% 反超 OpenAI 的 29%——Anthropic 用 1.34 亿 MAU vs OpenAI ~9 亿 MAU 实现 ARPU $16.20 vs $2.20。(TechCrunch / The Register)
  • OpenAI —— 4/30 TechCrunch：“对 Anthropic 限制 Mythos 大加批评后，OpenAI 自己开始限制 GPT-5.5 Cyber 访问”——5/1 起仅向"关键网络防御者"发放，能跑渗透测试 / 漏洞挖掘 / 恶意软件逆向。OpenAI 内部姿态从公开开放转向"高敏感能力 + 受限分发"，与 Anthropic 走到同一象限。(TechCrunch / NextGov)
  • Google DeepMind —— 4/29 财报：Alphabet 4 月股价 +34% 创 2004 年以来最佳单月；2026 capex 上调到 $180–190B；Workspace Updates 4/30 公告：Gemini in Sheets 5/6 起推送，70.48% 自主操作复杂表格成功率；Gemini 3.1 Pro 在 Deep Research Max 上将 BrowseComp 推到 85.9，比 Gemini 3 Pro 高 25+ 分。(CNBC / Workspace Updates / blog.google)
  • DeepSeek —— 4/29 TrendForce：Huawei Ascend / Cambricon / Hygon / 摩尔线程对 V4 完成 Day-0 适配，"模型即生态"速度过去仅属 Nvidia；4/26 gHacks：V4 部署内存需求 9.5× 下降；价格层面 V4 Pro 至 5/5 输入价 $0.036/M tokens（比 GPT-5.5 低约 10×）。(TrendForce / gHacks / Tom’s Hardware)
  • Meta —— 4/29 财报：2026 capex 上调到 $125–145B；股价盘后 −6%。Llama 4 Scout / Maverick 4/5 已发，Avocado / Mango 仍处 H1 2026 路线图状态、本日无新增 benchmark。(Fortune)
  • xAI —— Grok 4.3 Beta 仍处 4/17 发布后 SuperGrok Heavy 早期访问阶段；本日无新增前沿信号，4.3 全量 mid-to-late May 计划不变。(Artificial Analysis)
  • Mistral / Qwen —— Qwen3.6-27B（4/22）/ Plus（4/2）已发，本日无新增前沿模型；Mistral 本日无显著增量，留作下日 watchlist。(MarkTechPost Qwen3.6-27B)
  • 未验证 Frontier 信号 —— OpenAI 自研手机硬件 4/27 9to5Mac 单线披露，仍属 community 层；White House 起草联邦机构使用 Anthropic 方案 4/30 NextGov 单线，待第二信源。

Energy

  • Big Tech 2026 capex 总计 $650–725B：MSFT $190B（含 $25B 元件涨价）、GOOG $180–190B 并预告 2027 进一步显著增长、META $125–145B、AMZN ~$200B；CapEx 同比 +77%，单季度合并已 $130.65B。(CNBC Alphabet / Fortune / TheNextWeb / Yahoo Finance)
  • AI 用电与电网瓶颈：IEA 4 月初口径 2026 全球数据中心电力消费 ~1,100 TWh（≈日本全国）；Uptime Institute 称 AI 专属负荷将达 10 GW；美国电网互联排队周期已无法匹配超大规模商建排程。SMR 已锁定的"数据中心专属"核电承诺自 2024 末的 25 GW 累计扩张到 45 GW。(IEA / Data Center Knowledge)
  • 政策侧：White House 同时一边踩 Anthropic Mythos 扩张刹车、一边起草联邦机构 Anthropic 准入方案，"国家级 AI 基础设施"政策模糊性进一步抬升——能源 / 电网 / 国家安全三轴本周首次同步进入"跨域协调"语义。

Chips

  • Cambricon —— 4/29 Bloomberg：Q1 2026 营收 ~$423M、同比 +160%；上海股价单日 +14%；目标 2026 出货 50 万颗 AI 加速器（vs 2025 ~11.6 万），瓶颈在 SMIC 7nm 良率 ~20%。(Bloomberg / Tom’s Hardware)
  • Huawei Ascend 950 —— Invezz 4/29：DeepSeek V4 触发 BAT（ByteDance / Tencent / Alibaba）同时找 Huawei 加单 Ascend 950；Trendforce 4/29：Huawei + Cambricon + Hygon + 摩尔线程对 V4 完成 Day-0 适配。(Invezz / TrendForce)
  • Nvidia —— Rubin 平台已进入量产，partners 下半年起出货；NVIDIA 自有 quantum-AI 模型族 Ising 同步开源以对接 SMR / 量子混合 workload。(NVIDIA Newsroom Rubin / Ising)
  • TSMC —— Q1 净利同比 +58%、连续四季度创纪录；HPC（含 AI）占比 61%；A13 / N2U 工艺定档 2029，亚利桑那封测厂同步落地。(CNBC / The Next Platform)

Infra

  • AI 资本结构性回写企业利润：Fortune 4/30 以 Counterpoint 数据指出 Google / Amazon Q1 "AI 利润"中约一半来自其在 Anthropic 的股权增值（而非主营业务）——单季 Anthropic 估值 $300B → 接近 $900B 的轨迹直接反映在 GAAP / non-GAAP 报表项。(Fortune / Motley Fool)
  • Cloud / TPU 供给：AWS / GCP / Azure / OCI 列为 Vera Rubin 部署首批；Anthropic 同日扩大 Google Cloud TPU 与 Broadcom 直供（4/24 5GW Ironwood 协议加深）。(Anthropic)
  • Agentic 基础设施：blog.google 4/22 Deep Research Max（Gemini 3.1 Pro）首次原生支持 MCP，把异步 / 后台工作流作为 first-class；OpenClaw 4/30 GitHub stars 突破 30 万、保持 2026 增速最高项目地位（来自 Steinberger / TED 2026 主题演讲后的扩散）。(SiliconAngle / Fungies)
  • 未验证基础设施信号：OpenAI 自研手机硬件 4/27 9to5Mac 单线，White House 起草联邦机构 Anthropic 准入 4/30 NextGov 单线——两条均待第二信源。

Model

  • Anthropic Claude Opus 4.7（4/16 GA）—— 文档侧 4/30 仍在小幅迭代：xhigh effort 参数针对 coding / agentic 推荐；价格保持 $5/M in、$25/M out；Anthropic 公开承认仍落后内部 Mythos Preview 性能。(Anthropic / GitHub Changelog / Axios)
  • GPT-5.5 Cyber —— 4/30 起对"关键网络防御者"分发，能跑渗透 / 漏洞 / 恶意软件逆向；OpenAI 等同 Anthropic 进入"前沿能力 + 受限分发"。(NextGov / TechCrunch)
  • DeepSeek V4 Pro / Flash（4/24 preview，4/30 持续扩散）—— 1.6T MoE / 1M context / MIT，Day-0 在国产链全部跑通；MIT Technology Review 4/24 三大要点收口：“V4 在前沿能力上接近 GPT-5.5、价格仅约 1/10、彻底跑在中国硬件栈”。(MIT Tech Review / Bloomberg)
  • Gemini 3.1 Pro / Deep Research Max —— BrowseComp 85.9，比 3 Pro 高 25+ 分；Workspace Updates 4/30 确认 Gemini in Sheets 5/6 起推送（70.48% 自主操作率）。
  • Qwen3.6-Plus / 27B —— 4/2 Plus、4/22 27B，密集权重对标 397B MoE 在 agentic coding；Hugging Face / ModelScope 同步上架。(Alibaba Cloud / MarkTechPost)
  • arXiv 增量：AutoResearchBench（北京智源 4/28 trending，1,000 学术任务、Deep / Wide Research 双轴）— Hugging Face 4/28 trending 第一；MarketBench AI agent 作市场参与者评估；Efficient Benchmarking of AI Agents（HAL 9 benchmark + Terminal-Bench 2.0 系统化降本）。

Application

  • Novo Nordisk × OpenAI 战略合作：跨 drug discovery / 临床试验 / 制造 / 供应链 / 商业化全栈 AI 整合——大型生物制药首次以"全公司一次合并"姿态采用 OpenAI 模型。(showupandplay AI Daily 4/30)
  • Cadence × NVIDIA：Cadence 多物理仿真 + NVIDIA Isaac 机器人库 + Cosmos 开放世界模型，目标关闭 sim-to-real gap——数字孪生 + 物理 AI 进入工业落地的共识技术栈。
  • Agentic Commerce 三家姿态（DigitalCommerce360 4/30）：Anthropic / OpenAI / Google 的 agentic commerce 路径首次被并列报道——OpenAI 偏向 ChatGPT 内购物深度整合，Anthropic 走 Claude for Work + 企业代理嵌入，Google 走 Shopping Graph + Gemini 接管。(DigitalCommerce360)
  • 法律风险：佛州 AG James Uthmeier 4/30 宣布对 OpenAI 开启刑事调查（chat log 显示 ChatGPT 涉嫌指导嫌疑人选枪型 / 弹药 / 时机 / 地点）——AI "刑事级披露义务"自 4/23 加拿大 Tumbler Ridge 案后再升一级。

层间联动影响

  • 政府 ↔ 模型 ↔ 资本：白宫同日做"Mythos 暂停 70 家扩张 + 起草联邦 Anthropic 准入"两件事，意味着前沿能力扩散将以"国家许可"为节点；与之同步的 OpenAI GPT-5.5 Cyber 收紧使得高敏感能力的"白名单分发"成为新事实标准——而 Anthropic 的 $900B 估值正是建立在这种受限分发的稀缺性之上。
  • Energy ↔ Chips ↔ Infra：Big Tech $725B AI capex 上调 + Vera Rubin 量产 + DeepSeek V4 跨国产链落地三件事在同一周对齐，意味着 2026 算力供应将出现"美 Rubin / 中 Ascend 950"双栈并行——并把电网 / SMR 进度推到 strategic-asset 层级。
  • Model ↔ Application ↔ 资本：Anthropic Q1 收入超 OpenAI（31.4% vs 29%）+ Novo Nordisk 全栈采购 OpenAI + Anthropic 在 GOOG / AMZN 报表上贡献"AI 利润半数"——表明前沿模型变现已从"API 流量"切换到"企业一次性绑定 + 母公司股权"双轨。
  • 政策 ↔ 开源：Zig 4/30 反 LLM 贡献政策被 Anthropic 收购的 Bun 触怒——开源治理首次以 OSS / corporate-AI 边界形成结构性冲突；与同日 Cyber / Mythos 收紧并置后，"安全敏感能力"与"开源贡献身份"成为同一辩论的两端。
