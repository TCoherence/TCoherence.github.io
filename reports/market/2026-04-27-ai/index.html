AI 日报｜2026-04-27

一句话结论：Microsoft–OpenAI 第二阶段协议正式落地（Azure 主云延续到 2032、OpenAI 解锁多云、MSFT 不再向 OAI 支付分成、IP 许可改为非独占），与同日开庭的 Musk v. Altman OpenAI 治理庭审形成"商业绑定松、法庭对峙紧"的双线共振；与上周 DeepSeek V4 + Huawei Ascend 950PR 双主权栈、Anthropic Mythos / Glasswing 网安基线一起，AI 市场进入"算力栈、模型治理、商业架构同时重置"的关键周。

摘要

  • Microsoft–OpenAI 第二阶段协议公告（4/27）：OpenAI 产品默认仍 Azure 首发，但 OAI 现在可向任何云分发；Microsoft 对 OAI 模型/产品 IP 许可延长到 2032 但改为非独占；Microsoft 不再向 OpenAI 支付收入分成；OpenAI 向 Microsoft 的分成持续至 2030 但新增总额上限。市场解读为 OAI 进入"准独立运营"阶段，Microsoft 转向"主云 + 主要股东 + IP 许可方"三层捆绑。
  • Musk v. Altman OpenAI 信托案 4/27 加州奥克兰开庭：预计 4 周庭审，Musk/Altman 出庭，Nadella 与现任/前任 OpenAI 董事和资深研究员在证人名单。核心争点为 2017–2019 期间 OpenAI 由非营利向 capped-profit 改制是否构成对 Musk 的欺诈；2024 年公开邮件成 OAI 主要抗辩证据。
  • DeepSeek V4 / Huawei Ascend 950PR 双主权栈本周延续发酵：V4-Pro 1.6T MoE / 49B 激活、V4-Flash 284B / 13B 激活、上下文 1M token；价格 V4-Pro $1.74 / $3.48 per M tokens、V4-Flash $0.14 / $0.28；ByteDance $5.6B Ascend 950PR 订单 + Huawei 全年 750k 颗目标产能锁定供给。
  • Anthropic Claude Mythos Preview + Project Glasswing：Mythos 自主完成 FreeBSD 17 年龄 RCE（CVE-2026-4747）独立利用，Glasswing 联盟绑定 AWS / Apple / Google / NVIDIA / JPMC / Cisco / Broadcom / CrowdStrike / Linux Foundation / Microsoft / PANW；Opus 4.7 4/16 GA 价格不变但新 tokenizer 实测涨 0–35% 单 prompt 成本。
  • 应用层：Hermes Agent / OpenClaw 之争见分晓：Nous Research 的 Hermes Agent（持久记忆 + 闭环自学习）2 月发布以来 10 周 GitHub 110k stars，OpenClaw（Steinberger）3 月连爆 9 个 CVE / 12% skill 投毒率，开发者明显迁移；OpenAI Privacy Filter（4/22，1.5B / 50M 激活、Apache 2.0）同期成为 PII redaction 默认开源基线。
  • 能源：Meta 4/27 公开探索"太空太阳能"为 AI 数据中心供电（Bloomberg）；MIT/MIT-IBM 4/27 发布 AI workload 功耗预测工具。
  • 机器人：Sereact $110M Series B（4/27，Headline 领投），Cortex 2.0 在 BMW / Daimler / Bol / H&M 100+ 实地部署上训练，“动作前后果模拟”（predict-before-act）成为机器人 RL 新主线。

Frontier Labs / Frontier Model Radar

  • Microsoft × OpenAI 协议改写（OpenAI / Microsoft 4/27）：Microsoft 维持 OAI 主云身份至 2032、OAI 产品默认 Azure 首发；但 Microsoft 对 OAI IP 许可改为非独占、不再支付分成；OAI 可分发至任意云。市场解读：Azure 不再独占 OAI 推理流量是 GCP / AWS 中长期回血窗口；OAI 解除 Microsoft 单一捆绑后，2026H2 IPO 路径更清晰。(Microsoft 官方 / OpenAI 官方 / CNBC 4/27 / Bloomberg 4/27)
  • Anthropic Claude Mythos Preview + Project Glasswing（red.anthropic.com 4/20）：Mythos 自主识别并利用 FreeBSD 17 年龄 RCE（CVE-2026-4747），数千例 zero-day 已通过受信通道交付给 Glasswing 联盟（AWS / Apple / Broadcom / Cisco / CrowdStrike / Google / JPMC / Linux Foundation / Microsoft / NVIDIA / PANW），暂未对外开放普通 API。(red.anthropic.com / Schneier 4/20 / Foreign Policy 4/20)
  • Anthropic Claude Opus 4.7 GA（Anthropic 4/16）：定价 $5 / $25 per M tokens 不变，但新 tokenizer 在同样 prompt 上 token 数增长 0–35%，等效单调用账单上行；GitHub Copilot / Bedrock / Vertex / Foundry 同步可用。(Anthropic 官方 / GitHub Changelog 4/16 / Finout 拆解)
  • DeepSeek V4 Preview（DeepSeek 4/24）：V4-Pro 1.6T 总参 / 49B 激活、V4-Flash 284B / 13B 激活、上下文 1M token；本日（4/26–27）继续在 LMSYS / Artificial Analysis 榜单滚榜，"距前沿 3–6 个月"并把开源 SOTA 顶到接近 GPT-5.4 / Gemini 3.1 Pro 区间。(Simon Willison 4/24 / TechCrunch 4/24 / MIT Tech Review 4/24)
  • OpenAI Privacy Filter（开源）（OpenAI 4/22）：1.5B 总参 / 50M 激活、bidirectional token classifier、128k 上下文、Apache 2.0；PII-Masking-300k F1 = 96%。补齐 OAI 在"小模型 + 数据治理 + 本地推理"路线缺口。(OpenAI 官方 / VentureBeat 4/22 / Hugging Face)

Unverified / 观察名单：

  • xAI Grok 5 / 6T 参数 / Colossus 2 训练（Musk 个人发言，无第三方验证；保留在 watchlist）。
  • DeepSeek V4 GA 时间窗（DeepSeek 仍标 preview，没给 finalize 时间）。
  • Anthropic Mythos 公开 API 价位（Glasswing 联盟内 $25 / $125 per M tokens 是封闭定价，外部开放尚无窗口）。

Energy

  • Meta 探索"空间太阳能 → AI 数据中心供电"（Bloomberg 4/27）：在地面新增电力审批受阻 + 燃气调峰极限的背景下，Meta 把研究触手伸进 Space Solar Power（卫星阵列 + 微波下行）方向，意在为 2027–2030 周期的 AI 数据中心找新增量。(Bloomberg 4/27)
  • MIT 发布 AI workload 功耗预测工具（MIT News 4/27）：MIT-IBM Watson AI Lab 用预测模型替代 GPU 运行时实测，目标在数据中心调度阶段就预估 AI 任务功耗，缓解 2030 前数据中心占美国总用电 12% 的物理压力。(MIT News 4/27)
  • IEA 数据中心年报（IEA 2026）：2025 年数据中心耗电同比 +17%，AI 数据中心耗电增速远超 +17%，全球电力总需求 +3% 衬托下"AI 单一行业拉电"基本定型；2030 前数据中心耗电预计翻倍、AI 部分预计翻三倍。(IEA)
  • 美国电费传导：AIC 4/26 报告，部分受影响地区居民电费两年内接近翻倍；Fortune 4/20"美国新增电力一半流向数据中心"持续被引用。(AIC 4/26 / Fortune 4/20)

Infra

  • Microsoft–OpenAI 协议改写最大 infra 含义：Azure 失去对 OpenAI 工作负载的事实独占 → AWS / GCP / Oracle 都进入"OAI 推理可分流"窗口；Microsoft 不再向 OAI 支付分成 → MSFT 自有云 capex 节奏更可控、分摊到 Copilot / Foundry 路线的 ROI 测算更直接。(CNBC 4/27)
  • Anthropic 算力多线绑定：W17 已确认 Google $40B / 5GW Ironwood + Amazon $25B / 5GW Trainium 双管，本周 Anthropic 进一步与 Google + Broadcom 公布合作，主线是"自研 ASIC（Trainium / Ironwood）+ 多云冗余"，把 NVIDIA 单一依赖度往下压。(Anthropic 官方)
  • 机器人专用云 / 实地数据闭环成为新 infra 形态：Sereact Series B 后将把 Cortex 2.0 服务规模从 100+ 部署线扩展到美国市场；BMW / Daimler / Bol / H&M 真实拣货数据成训练集，是少数把"工业现场操作流"做成 RL 数据飞轮的西方机器人公司。(Bloomberg 4/27 / TNW 4/27)
  • Hyperscaler AI capex 仍在加速：MSFT / META / GOOG / AMZN 2026 合计 AI infra 投放正向 $700B 量级走，但本日 Microsoft 协议变化使"超大规模 AI capex 的边际 ROI 假设"被重新审视。

Application

  • Hermes Agent vs OpenClaw 攻守易位：Nous Research Hermes Agent（持久记忆 + 闭环自学习）2 月发布、10 周 110k GitHub stars；OpenClaw（Steinberger）3 月连爆 9 个 CVE（含一个 CVSS 9.9）+ ClawHub 2,857 entries 中扫出 341 个恶意 skill（≈12%）后，开发者明显往 Hermes 迁移；"Agent 能否记住 / 学习"成为 2026 年开源 agent 的分水岭。(New Stack / Nous Research Hermes 仓库 / shareuhack 4/22)
  • OpenAI / Anthropic / Google / Zapier "agent 平台化"四发齐至：GPT-5.5 agent mode 进 ChatGPT 下拉框（4/24）；Anthropic Claude Managed Agents $0.08/session-hour + tokens；Google Workspace Studio（4/22）允许业务用户在 Gmail/Docs/Sheets/Drive/Meet/Chat 自然语言搭 agent；Zapier 4/23 扩张 MCP + 治理控制——"Agent 从 demo 到 SMB infra"门槛在 4 月被一次性压平。(blog.mean.ceo)
  • Sereact Cortex 2.0 进入工业级 RL 数据飞轮：BMW / Daimler / Bol / H&M 实地拣货数据 → 训练 → 上线 → 修正闭环；为机器人侧"端到端模型 + 真实数据"路线提供西方阵营的标杆案例。
  • 中国 ToB 落地：本日《十字路口》节目记录百融智能（港股 1600 人）在风控审批 / 联络中心 / 招聘面试 / 财务报销 / 合同审核 5 个企业级 Agent 生产场景的部署路径；"概念→责任人→闭环可衡量场景"被作为传统企业 ToB 落地的可复制模板。(小宇宙 4/26)

层间联动影响

  • Energy ↔ Infra：Meta 跨级别尝试空间太阳能 → 反映地面电力 + 选址 + 监管已经构成 2027 后 AI 数据中心扩张的硬约束；MIT 新功耗预测工具是同一约束的另一侧"软切口"，两者一起把 AI capex 向"先算电、再选址、再排集群"前置化。
  • Chips ↔ Model（中国侧）：DeepSeek V4 + Huawei 950PR + ByteDance / Ali / Tencent 大单 = 中国主权算力栈"芯片-模型-推理-应用"四段闭环已基本接通；与 W17 周报里"美国栈 = OAI / Anthropic / Google / Amazon 各自绑大算力"形成镜像结构。
  • Model ↔ Application：Hermes vs OpenClaw 的攻守易位 + Mythos / Glasswing 的网安联盟绑定 = "记忆 + 安全 + 合规"成为本季 Agent 框架竞争的新三轴；过去半年的"工具调用 + 多步规划"赛道已经被这三轴覆盖。
  • Infra ↔ 商业架构：Microsoft–OpenAI 协议变化 + Musk 庭审同日开庭，把 AI 商业架构从"独占绑定 + 私下博弈"推向"非独占 + 法庭公开博弈"；这对 OAI 估值锚（IPO / 持股结构）与 Azure / GCP / AWS 竞争格局是连锁反应。
  • Application ↔ 治理：Tumbler Ridge / Walden / Mythos 三件事叠加，把"AI 内部告警 → 法律强制通报"做成本季最具象的立法窗口；Anthropic 用 Glasswing 这种"封闭联盟"方式部分自我监管 + 自我披露，把治理标准的事实参考点拉高。
