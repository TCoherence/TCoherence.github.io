AI 日报｜2026-04-25

一句话结论：DeepSeek V4 在 Huawei Ascend 上原生跑出前沿性能、Google 加注 Anthropic 至 $40B 并独家供应 5GW Ironwood TPU、OpenAI GPT-5.5 全量铺开——同一周内中美两套算力栈同时完成"主权化"与"超大规模化"，Agent 时代的产能与生态分裂正在定型。

摘要

  • 模型层（mainline）：DeepSeek V4 Pro/Flash 4/24 预览发布（1.6T MoE / 1M context / MIT 协议），CSA+HCA 注意力将推理 FLOPs 降 73%；GPT-5.5 同日全量推送 ChatGPT/Codex 与 API。
  • 基建层（mainline）：Google 4/24 宣布对 Anthropic 总额最高 $40B 投资（$10B 即时 + $30B 里程碑），承诺 5GW Ironwood TPU 算力；Anthropic Mythos 通过 Project Glasswing 进入 Amazon/Apple/Cisco/Microsoft 等关键基础设施分发。
  • 芯片层（mainline）：Huawei 宣布 Ascend AI 超节点"全面支持"DeepSeek V4，香港 SMIC +9%、Hua Hong +15%；NVDA 同日刷新历史新高，市值首次站上 $5T。
  • 人物 / 社区：Sam Altman 4/23 对加拿大 Tumbler Ridge 屠杀社区公开道歉，把 ChatGPT 安全干预首次推到刑事级公共问责面前；张小珺 4/24 上线 3.5 小时罗福莉访谈，定义"Agent 范式吃后训练"叙事。
  • 本日候选池新增 2 名：罗福莉（Xiaomi MiMo 负责人）、梁文锋（DeepSeek 创始人 / V4 共同作者）。

Frontier Labs / Frontier Model Radar

主线（高置信）

  • DeepSeek V4 Pro / V4 Flash 预览发布（4/24）：Pro 1.6T 总参 / 1M context，Flash 284B / 1M context，均 MIT 协议开源；自研 Hybrid Compressed Sparse Attention（CSA）+ Heavy Compressed Attention（HCA），推理 FLOPs 降 73%。Pro 定价 $0.145/M 输入 / $3.48/M 输出，Flash $0.14 / $0.28。论文由 CEO 梁文锋共同署名，强调"国产闭环"——首个完全跑在国产链上的前沿模型（CNBC, SCMP, Fortune, TechCrunch）。
  • GPT-5.5 / GPT-5.5 Pro 全量推送（4/24）：同时进入 ChatGPT Plus/Pro/Business/Enterprise、Codex 与 API；Pro 仅在 ChatGPT Pro/Business/Enterprise；强调"长程任务自主完成 + 工具编排"。同日上线 ChatGPT Images 2.0（含"Images with Thinking"，用更长推理时间精炼图像）。Fortune 评论：模型发布频率已"接近软件更新节奏"（OpenAI, Fortune, Releasebot）。
  • Google → Anthropic $40B 投资 + 5GW Ironwood TPU 多吉瓦协议（4/24）：$10B 现金按 $350B 估值即时入资，$30B 里程碑触发；首期 400k TPUv7 Ironwood（约 $10B 整柜，Broadcom 直供 Anthropic），余下 600k 通过 GCP 租用（约 $42B 收入口径）。NextWeb 直接挑明动机：Claude 在企业市场销量已超过 Gemini，Google 内部组建"strike team"专攻 Gemini 编码能力补短板（CNBC, TechCrunch, Anthropic 官方, Data Center Knowledge, The Next Web）。

次线 / unverified frontier signals

  • Anthropic Mythos / Project Glasswing 持续扩散：Kemba Walden（前美国国家网络总监）4/23 在 Fortune 专栏首次以前政府高官身份警告 Mythos 网络攻击能力（zero-day 复现率 83.1%），呼吁协调关键基础设施防御投资（Fortune, Anthropic 红队官页）。Project Glasswing 合作方覆盖 Amazon/Apple/Broadcom/Cisco/CrowdStrike/Linux Foundation/Microsoft/Palo Alto Networks。
  • Xiaomi MiMo-V2.5 系列（4/22）：Luo Fuli 团队推出 V2.5 推理模型 + V2.5-Pro 全模态 Agent（公测中、即将开源）+ V2.5-TTS / V2.5-ASR 待发；4/24 张小珺播客 3.5 小时访谈系统化输出 Pre-train→Post-train 范式叙事（VentureBeat, 小宇宙播客）。
  • 未交叉验证信号：xAI 与 Mistral 本日无显著增量；Meta Avocado/Mango 仍处于"H1 2026 路线图"状态、未见公开 benchmark。

关键人物与社区信号

  • Sam Altman（@sama）— Tumbler Ridge 公开道歉信（4/23 发出，4/24 经 BC 省长 Eby 在 X 公开）：承认 6 月已 ban 该账号但"未通知执法"；ChatGPT 滥用检测命中后无强制升级机制成为质疑焦点。这是 OpenAI 首次因刑事案被 CEO 个人级公开问责，对 Trust & Safety / 法律披露义务讨论是高信号事件（CBC, CNN, CBS）。
  • Liang Wenfeng（梁文锋，DeepSeek CEO）：作为 V4 论文（CSA+HCA）共同署名作者亲自上场，与 Huawei / Cambricon 合作公开宣告"国产硬件 + 开源前沿"路线；DeepSeek 同步加大开源治理与"普惠定价"叙事，与 OpenAI"软件化迭代"形成路线分叉（36kr）。
  • Luo Fuli（罗福莉，Xiaomi 大模型团队负责人 / 前 DeepSeek）：4/24 张小珺商业访谈录 3.5 小时长访（首次长访谈），系统讲清"Pre-train 主导 Chat → Post-train 主导 Agent"范式跃迁、卡的分配、“组织平权”——把 Claude Opus 4.6 / OpenClaw 引发的"AI 巨震"做了结构化解读（小宇宙播客）。
  • Andrej Karpathy（@karpathy）：本周延续"LLM Wiki / 第二大脑"叙事——把 raw 资料丢给 LLM 自动生成知识库与反向链接，逐步替代"AI 写代码"的主用例；4/24 多家媒体围绕这一方向继续放大（Medium - Neural Notions, Simon Willison tag）。
  • Kemba Walden（候选池）：见上文 Mythos 段落，Fortune 专栏成为本周"前政府高官公开断言 Mythos 攻击力"的首发渠道。
  • 唐文斌（候选池）：4/24 晚点聊播客视频版上线，复盘旷视并公布原力灵机"不做人形机器人"的差异化具身路线（小宇宙）。

Energy

  • Bloom Energy ↔ Oracle 2.8GW 燃料电池协议：4 月签署、首期 1.2GW 已部署、其余 12 个月内交付，用于支撑 Oracle AI 数据中心；Bloom 90 天内累计签下 $76.5 亿数据中心订单——AI 算力开始把分布式燃料电池从备电拉成主电（ITIF, AI Power Weekly）。
  • EIA 上调 2026 / 2027 电力需求：2025=4,195 → 2026=4,260 → 2027=4,388 (B kWh)，主因 AI 数据中心；若兑现将创 2007 年来首次"四年连涨"。Energy Sec. Wright 已要求 FERC 4/30 前回应大用户与新建电源联合并网快速规则（同上 ITIF / IEA）。
  • 数据中心 vs 电网瓶颈持续：Omdia 等口径预计 2026 全球 hyperscaler capex >$600B，其中 ~75%（≈$450B）直接挂在 AI 基建；单 GB200 NVL72 机柜功耗 120-140kW，已强制液冷 + 现场升级供配电（Data Center Knowledge, Data Center Frontier）。
  • 高置信信号有限：本日没有新的核电或 SMR 突破公告；继续观察 Meta 6.6GW 三笔核电协议执行节奏。

Chips

  • DeepSeek V4 ↔ Huawei Ascend 950PR + Cambricon 协同（4/24）：Huawei 官方"全面支持"V4，Ascend 超节点完成 V4 推理适配；港股反应剧烈：SMIC +9%，Hua Hong +15%——市场把 V4 视作"国产链可用"的实证而非鼓动（Investing.com, Huawei Central, Phemex）。
  • NVDA 4/24 收盘 $208.27（+4.32%），市值首次破 $5T：Intel 数据中心营收 +22% 拉动情绪，市场认定 AI 推动 CPU 也复苏；下次财报 5/20，预期同比 EPS 三位数增长（Motley Fool, Benzinga）。
  • Google Ironwood TPUv7（前一日）：八代分裂为训练 / 推理两条线，目标 TSMC 2nm；本次与 Anthropic 协议把 Ironwood 推到首个"百万级"对外承诺规模（The Next Web - Ironwood, SemiAnalysis）。
  • Cambricon 2026 产能展望未变：仍指向 ~50 万片（300k Siyuan 590/690 走 SMIC N+2 7nm），但 ~20% 良率仍为硬约束；HBM 供应仍紧——本日上涨更多反映 V4 的需求侧确认（Tom’s Hardware）。

Infra

  • Google → Anthropic 5GW Ironwood TPU 多吉瓦协议（mainline）：见上文。从基建视角看，相当于 Anthropic 锁定一组与 OpenAI / Microsoft 拓扑独立的算力供给（首批 400k TPUv7 整柜由 Broadcom 直供 + 600k 走 GCP 租用），2027 年起逐步上线（Anthropic 官方, Data Center Knowledge）。
  • MCP 安全治理仍未落地：4/16 OX Security 披露的 STDIO 架构级漏洞、20 万+ 服务器、1.5 亿次 SDK 下载至本日仍未见上游协议补丁；Anthropic"设计如此"立场未撤回——基建层风险显性化。
  • 企业 AI 平台基础设施成型：Salesforce Headless 360（4/16）把 CRM/服务/营销/电商全栈以 API+MCP+CLI 暴露给 Claude Code/Cursor/Codex；Databricks Unity AI Gateway（4/15）把 Unity Catalog 治理模型扩展到 Agent 与 MCP 调用（FifthRow 综述）。

Model

  • 三家旗舰同周交付：GPT-5.5（4/24）、Claude Opus 4.7（4/16）、DeepSeek V4 Pro/Flash（4/24）——三条产品线在同一周内把"长程任务、Agent 编排、1M context"推到生产口径；Mythos Preview 仍由 Anthropic 持有不公开发，构成"能力分发分级"实例。
  • 路线分叉清晰化：
    • OpenAI → “软件化迭代”（Fortune 评 GPT-5.5 像"软件更新"）；
    • Anthropic → “能力 + 安全分层”（Mythos 红队封顶 + Opus 4.7 商用 + Glasswing 选择性下放）；
    • DeepSeek → “前沿开源 + 国产硬件原生 + 普惠定价”，国产链闭环成形。
  • MiMo / Qwen 等中国阵营：Xiaomi MiMo-V2.5（4/22）开始走"全模态 Agent"路线；Qwen 3.6 Plus（4/2）已落地 Hybrid Linear Attention + Sparse MoE。

Application

  • ChatGPT Images 2.0 + Images with Thinking（4/24）：所有 ChatGPT 计划可用，付费层多 “thinking” 模式；同步上线"对话式购物 + 图像相似商品 + 并排比价"，把多模态 + 工具调用打通到消费场景。
  • Salesforce Headless 360（4/16）：把 27 年沉淀的 CRM/Service/Marketing/Commerce 全栈 API 化、MCP 化、CLI 化——Claude Code / Cursor / Codex 直接当"无浏览器自动化层"。
  • 企业 Agent 治理走向生产口径：FifthRow 综述指出 4 月已是"从孤立 PoC 到合规级生产部署"的拐点，EY、Salesforce、JPMorgan 公开了多 Agent 编排到关键流程的案例（FifthRow）。
  • GitHub trending（截至 4/25）：OpenClaw 已破 21 万 stars 继续扩散；google/adk-python（8.2k+）、meta-llama/llama-stack（6.4k+）、openai/codex-cli（5.8k+）为本月新晋 trending；Superpowers 仍稳居 Claude Code 生态社区插件第一（blog.bytebytego, GitHub trending）。

层间联动影响

  • Chips ↔ Model ↔ Infra（中国闭环）：DeepSeek V4 + Huawei Ascend 950PR + Cambricon Siyuan 形成"前沿模型在国产链上原生跑通"的首例；港股 SMIC/Hua Hong 当日大涨即是一次性"投票"。其后果是中美前沿模型推理栈进一步去耦——出口管制的"封顶效应"在生态侧被部分中和。
  • Infra ↔ Model（美国超大规模化）：Google 把 Ironwood 大规模租给 Anthropic，本质是用"硬件供给绑定"补 Gemini 在企业市场的销量短板；OpenAI 同周用"软件化迭代频率"维持感知；硬件供给关系正在替代纯模型能力作为长期竞争壁垒。
  • Model ↔ Application（Agent 范式落地）：GPT-5.5 / Opus 4.7 / V4 三模型 + Salesforce Headless 360 / Databricks Unity AI Gateway 的组合，让"Agent 范式吃后训练"（罗福莉访谈语）从研究叙事变成可买卖的产品堆栈。
  • Energy ↔ Infra：Oracle 2.8GW Bloom 燃料电池 + Chevron-MS 直供天然气 + Meta 6.6GW 核电协议反映"自带电源"已是 hyperscaler 的默认选项；电网并不是支撑下一波算力的唯一依赖。
  • People ↔ Governance：Altman Tumbler Ridge 道歉信将"AI 公司是否有义务向执法机关上报暴力意图"的讨论从政策圈推到刑事级公共辩论；与 Mythos 网络武器化讨论叠加，AI 治理已从能力评估扩展到事件响应义务。
