5-Layer Macro News — 2026-05-04

Energy

  • Meta 已与 Vistra / TerraPower / Oklo / Constellation 签约，合计承诺 ~6.6 GW 核电；PJM 区 1.2 GW 校园 2030 上线，是 frontier-lab 在能源端最深的卡位。(Meta)
  • PJM 4/29 重启电源接入流程，811 个新发电项目排队申请并网；其中 Commonwealth Fusion Systems 申请在维州建首座核聚变电厂（早期 2030s 投运）。(Inquirer)
  • PJM 警示 2027 年起最多 60 GW 供应缺口可能导致区域停电，2026 hyperscalers 计划在数据中心上花费 ~$700B。(Belfer Center)

Chips

  • NVIDIA Vera Rubin 平台进入全量生产（CES 2026 公布的 7 颗新芯片），Rubin GPU + Vera CPU + NVLink 6 + ConnectX-9 + BlueField-4 + Spectrum-6 + 首次集成 Groq 3 LPU；H2 2026 落地 AWS / GCP / Azure / OCI 与 CoreWeave / Lambda / Nebius / Nscale。NVIDIA 估计 Rubin 运行成本是 Blackwell 的 1/10。(NVIDIA Newsroom / Data Center Knowledge)
  • OpenAI × NVIDIA 战略协议：H2 2026 落地首批 1 GW Vera Rubin 系统，每部署 1 GW 触发 NVIDIA 累进投资，最高 $100B。(NVIDIA Newsroom)
  • Pentagon 5/1–5/4 IL6/IL7 协议把 NVIDIA 与 OpenAI / Google / Microsoft / Amazon / Oracle / SpaceX / Reflection AI 同批纳入分类网络，进一步绑定 NVIDIA 在 AI 基础设施侧的卡位。(CNN)

Infra

  • Pentagon 5/1–5/4 IL6/IL7 协议是本周最大的 infra 主线：8 家厂商进入分类网络（OpenAI / Google / Microsoft / Amazon / Oracle / NVIDIA / SpaceX / Reflection AI），通过 GenAi.mil 平台对 130 万国防人员开放。「防 AI 厂商锁定」是国防部明示的架构原则。(Tom’s Hardware / Breaking Defense)
  • Anthropic × Google × Broadcom 三方算力扩盘：Anthropic 拿到约 3.5 GW 算力承诺（内含 TPU + Broadcom 定制硅）；与 4/24 Google 宣布最高 $40B 投资 Anthropic（首期 $10B / $35B 估值 / 含 5GW 算力）一脉相承。(Anthropic / Bloomberg)
  • Q1 2026 全球风险投资创纪录 ~$300B，AI 占 80%（$242B）；AI 基础设施类 25 笔披露融资合计 $10.64B（中位 $200M，已是产业资本规模而非传统 VC 规模）。(Crunchbase)

Model

  • Anthropic：ARR 突破 $30B（5 个月内 $9B→$30B），$1M+/年企业客户从 ~500 翻至 1000+；同周被 Pentagon 显式排除 IL6/IL7。新一轮融资讨论估值 $900B（CNBC 单源）。Claude Opus 4.7 于 4/16 GA。(PYMNTS / CNBC 估值 / GitHub Changelog)
  • OpenAI：4/23 GPT-5.5 发布（SWE-bench Pro 58.6% / Terminal-Bench 2.0 SOTA 82.7%），4 月底推出 GPT-5.5-Cyber + Cybersecurity Action Plan；Spud 即 GPT-5.5，仍属 GPT-5 家族，GPT-6 公开窗口未出。(Nextgov / felloai)
  • Mistral 4/29 上架 Medium 3.5，是本周开源端唯一 frontier 主线新版本。(Codersera)
  • 论文层（来自 paper-digest）：Web2BigTable WideSearch SOTA 38.50 / XBench-DeepSearch 73.0；OSCAR 首次把 MCTS 接入 DPO 数据构造；Stable-GFlowNet 给 LLM red-teaming 训练栈提供新稳定性基线。

Application

  • Microsoft Agent Framework 1.0（4/3 GA）：production-ready OSS，支持 .NET / Python，原生集成 MCP，含 Magentic-One 多 agent 编排；与 LangGraph / CrewAI 形成主流 agent 框架三角。(Microsoft)
  • Lukilabs Craft Agents（5/2，Apache-2.0）发布，是本周 application 层 agent 框架侧唯一 OSS 第三方新增。(AIToolly)
  • Claude Code 5 月更新：智能 model 选择、project purge 工具、ANTHROPIC_BEDROCK_SERVICE_TIER 环境变量、PR URL → /resume session 索引、Bedrock 429 误用 status.claude.com 修复。(Claude Help)
  • Pentagon × GenAi.mil：8 家厂商分类网络部署直接面向 130 万国防人员，已构建数十万 AI agent；application 层最大单一企业级用户场景。(Breaking Defense)
  • 论文层（来自 paper-digest）：fleet-scale RL 通用机器人 95% 成功 给「机器人版 RLHF」提供边部署边训练范式；Skill SSL 表征 是 Agent skill 市场化复用的方法论先决条件。

层间联动

  • NVIDIA Vera Rubin（chips） → Pentagon IL6/IL7 + OpenAI 10 GW（infra）：H2 2026 首批 1 GW OpenAI Vera Rubin 系统投产，与 5/1 Pentagon 协议同期落地，意味着 NVIDIA 在「军用分类网络 + 超大商业部署」两端同时进入主线。NVIDIA 累进投资 OpenAI 最高 $100B 把 chips→infra→model 三层串成同一资本结构。(NVIDIA Newsroom / CNN)
  • PJM 60 GW 缺口（energy） → 5/1 Pentagon 8 家 + hyperscalers $700B 数据中心 capex（infra）：能源缺口 + infra 资本同时放大，给 Meta / Oklo / Constellation 等核电卡位提供了「能源即基础设施竞争力」的具体落地点。(Belfer Center / Inquirer)
  • Anthropic Mythos 安全护栏（model） → Pentagon 排除（infra） vs OpenAI GPT-5.5-Cyber 入选（model+application）：同一周 Anthropic 因维持军用安全护栏被 Pentagon 排除，OpenAI 反向推出 GPT-5.5-Cyber 抢占国防安全应用，模型层的安全姿态直接转化为 infra/application 层的市场份额分化。(CNN / Nextgov)
  • Reflection AI Pentagon 资格（model+infra）：未公开发布商业 LLM 即获 IL6/IL7，Pentagon 在 frontier-startup 上做了与 NVIDIA $2B 同方向的押注；意味 chips → 创业层 model 的资本传导链条形成。(Tracxn)
  • Web2BigTable 学术 SOTA（model） → Google Deep Research Max + OpenAI Deep Research（application）：开源/学术给 deep-research agent 设了第一条公开 SOTA 基线，给 application 层 deep-research 类产品提供了可复现的对比锚。(arxiv 2604.27221 / Google Blog)
