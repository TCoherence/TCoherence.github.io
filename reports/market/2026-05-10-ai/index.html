AI 日报｜2026-05-10

一句话结论：5/9–5/10 周末窗口主线由 Anthropic Claude Mythos Preview / Project Glasswing + Opus 4.7 GA + Google $40B 注资三轴定调——frontier 实验室竞赛进入 “新一代旗舰 + 算力深绑 + PE 渠道分销” 的并列闭环阶段；同期能源端 Riot-Terrestrial 4 GW IMSR 与 IREN-NVIDIA 5 GW 部署在同一周交付，把 SMR 从 PR 概念推到 GW 级 take-or-pay 真实电力侧解；OSS 阵营 Moonshot Kimi K2.6、智谱 GLM-5.1 与小米 MiMo-V2.5 把 agentic coding 当作开源主线，但 paper-digest 今日 stale，论文层缺一手数据。

  • frontier_radar：Anthropic 5/9 公布 Claude Mythos Preview + Project Glasswing 受控开放并伴随未授权访问调查，Opus 4.7 GA 与 Pro/Max/Team Claude Code 速率上限翻倍同步落地；OpenAI 推 ChatGPT Trusted Contact 自伤防护与 Memory 升级；Google DeepMind 上 Workspace Intelligence + Deep Research Max；xAI Grok 4.3 + STT/TTS API 进入扩散；Meta、Mistral、Qwen、DeepSeek 48h 窗口无新一线发布。
  • paper_layer：paper-digest 2026-05-10.json 缺失（最新仅到 5/8，stale），未消费具体论文；补 3 条 paper-digest 不覆盖技术信号——NVIDIA Alpamayo 1 + AlpaSim 自动驾驶开源栈、智谱 GLM-5.1 + 小米 MiMo-V2.5 开源放量、GitHub trending 偏 agent eval / 编排（jcode、Ruflo）。
  • people_pool：tracked 池本日 7 人有信号——Simon Willison（Code w/ Claude live blog + Heavybit 播客）、Mira Murati（Nemotron Coalition）、Alexandr Wang（Anthropic/Meta 算力对标锚点）、Yang Zhilin（Kimi K2.6 GA + Vendor Verifier）、Dhanji Prasanna（Goose / MCP 引用回升）、Karol Hausman（PI π0.7 持续）、Sam Altman（GPT-5.5 / Images 2 节奏）；新增 3 名候选——Jon Gray（Blackstone, AI 分销资本）、Zhuofeng Li（TIGER-Lab agentic retrieval 一作）、Lukilabs（Craft Agents OSS GitHub trending）。
  • 5-layer：energy（Terrestrial-Riot 4 GW IMSR + 美 AI DC capex $1.4 万亿）、chips（Jensen “horrible outcome” + Huawei Ascend 950PR/950DT FP4 ~2.8x H20 + AMD MI308 Q1 中国销售腰斩）、infra（IREN-NVIDIA 5 GW $3.4B 预付 + Google $40B 注资 Anthropic + Oracle-Bloom 燃料电池）、model（Meta Avocado 推迟 + Gemini 3.2 Flash 泄露 + Claude Mythos Preview 进入 5 月旗舰榜）、application（Perplexity Computer for Enterprise + Anthropic-Blackstone-H&F-Goldman $1.5B 服务公司 + Moody’s/Microsoft 365 集成）。
  • cross-layer：energy↔infra↔chips（SMR 与 GW 级 GPU 部署同周对齐）；model↔application↔infra（80x 收入 → $40B + Colossus 1 + PE 嵌入式工程师四段闭环）；chips↔model（NVIDIA 把 frontier 首发 Ascend 视为美国国家级风险）。

Frontier Labs / Frontier Model Radar

5/9–5/10 frontier 主线由 Anthropic 与 OpenAI 主导：Anthropic 公布 Claude Mythos Preview + Project Glasswing，把自主漏洞挖掘作为下一代旗舰能力并伴随未授权访问风波；OpenAI 推出 ChatGPT Trusted Contact 安全功能与 Memory 升级。Google DeepMind 持续推 Workspace Intelligence 与 Deep Research Max，xAI 处于 Grok 4.3 + 语音 API 发布后的扩散期。Meta、Mistral、Qwen、DeepSeek 48 小时窗口内无新一线发布，停留在 Llama 4 / Medium 3.5 / Qwen3.6 / DeepSeek-V4 Preview 的生态推进阶段。

OpenAI

  • ChatGPT Trusted Contact 自伤防护（5/9）：成年人可指定一名信任联系人，系统判定严重自伤风险时收到通知；面向个人账户、不开放给 Business/Enterprise/Edu 工作区。TechCrunch / gHacks。
  • ChatGPT Memory 升级（5/9）：Plus/Pro 跨过往对话、saved memories、文件、Gmail 拉取上下文；引入 memory sources 让用户看到并编辑触发本次回答的记忆来源。Releasebot / Phil’s Tech。

Anthropic

  • Project Glasswing + Claude Mythos Preview（5/9）：未发布 frontier 模型受控开放给 40+ 关键软件组织，承诺最高 1 亿美元 usage credits + 400 万美元开源安全捐赠；官方称 Mythos 自主发现并利用 FreeBSD NFS 17 年历史 RCE 漏洞 CVE-2026-4747。Anthropic Mythos / letsdatascience。
  • Mythos 未授权访问调查（5/9）：第三方承包商账户被指在私密论坛分享访问，Anthropic 已展开调查；部分安全研究者对漏洞挖掘能力的独占性持保留意见。Releasebot / letsdatascience。
  • Claude Opus 4.7 GA + 速率上限翻倍（5/9）：Pro/Max/Team/Enterprise 的 Claude Code 与 API 速率上限翻倍并移除 Pro/Max 高峰削减。Releasebot / Releasebot Claude。

Google DeepMind

  • Gemini Workspace Intelligence（5/9）：Gmail/Chat/Calendar/Drive 实时上下文打通进入工作区 AI 流程，配合 Daily brief、imported chats 把 Gemini 推向日常工作底座。mean.ceo blog / Releasebot Gemini。
  • Deep Research Max（5/9）：下一代 Gemini 自主研究 agent，把多步检索 + 长文综合往自治研究方向继续推进。Google Blog。

Meta

  • 5/9–5/10 无新一线发布：主线仍是 Llama 4 Scout/Maverick（17B active MoE 原生多模态，Scout 10M 上下文、Maverick 对标 GPT-4o），LlamaCon 给出的 Behemoth 288B active 预览路线持续。Meta AI / LlamaCon。

xAI

  • Grok 4.3 发布余波（5/9）：1M 上下文 + 原生视频输入 + 内建推理，定价 1.25 美元/M 输入 token；Intelligence Index 53、CaseLaw v2 与 CorpFin 第一、GDPval-AA 相对 4.20 +300 Elo。NLPlanet via Medium / xAI 官博。
  • Grok STT/TTS API（5/9）：复用 Grok Voice/Tesla/Starlink 客服底层语音栈，向开发者开放。xAI 官博 / Releasebot xAI。

Mistral

  • 5/9–5/10 无新发布：主线为 4/29 上线的 Mistral Medium 3.5（合并 Medium 3.1 通用、Magistral 推理、Devstral 2 agentic coding 三条产品线）与持续推进的欧洲数据中心扩张。codersera / Mistral 官博。

Qwen

  • 5/9–5/10 无新模型上线：最近节点为 4/17 Qwen3.6-35B-A3B（35B/3B active MoE，agentic coding 取向）与 4/24 Qwen3.6-27B 稠密多模态。codersera / fazm.ai。

DeepSeek

  • 5/9–5/10 无新模型发布：最近节点为 4/24 DeepSeek-V4 Preview（V4-Pro 1.6T/49B active、V4-Flash 284B/13B active，1M 上下文，Pro 在数学与代码超过所有开源模型，仅世界知识落后 Gemini 3.1 Pro），社区追踪 V4.1 节奏 + 7B 融资带来的开源权重加速。TechCrunch / MIT Tech Review / dasroot.net。

未验证 Frontier 信号

  • Mythos Preview 第三方账户外泄（5/9）：BBC/Bloomberg/CNBC 报道通过第三方承包商账户在私密论坛二次外泄；Anthropic 仍在调查中，结论未定。Tom’s Hardware 等媒体对漏洞挖掘能力的独占性主张持质疑——作为 rumour 分类，待官方调查结果跟进。letsdatascience。
