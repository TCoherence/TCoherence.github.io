AI 日报｜2026-04-28

一句话结论：Microsoft-OpenAI 排他协议正式终结、AGI 触发条款"安静下葬"，Anthropic 同周接连吸纳 Google 400 亿与 Amazon 250 亿合计 650 亿美元，DeepSeek V4 开源端冲击 1M 上下文成本曲线，前沿赛道在"超大模型 + 超大资本 + 解绑云"的三轴下加速重排。

摘要

  • Microsoft × OpenAI：4 月 27 日双方宣布修订商业协议，Microsoft 失去对 OpenAI IP 的独家许可（保留至 2032，但非独家），OpenAI 可在 AWS / Google Cloud 等任意云上部署产品；20% 营收分成保留至 2030 但加总额封顶；曾决定双方关系终局的"AGI 触发条款"被整体删除（Microsoft 官方、CNBC、Simon Willison）。
  • Anthropic 双轮注资：Google 在 4 月 24 日宣布最高 400 亿美元投资 Anthropic（以 100 亿美元起跳，350 亿估值，含 5GW 算力承诺）；本周末再叠加 Amazon 此前确认的最高 250 亿美元加码——Anthropic 同时锁定 AWS Trainium 与 Google TPU 双供应链（TechCrunch、Motley Fool）。
  • DeepSeek V4 开源：4 月 24 日 DeepSeek-V4-Pro（1.6T MoE / 49B 激活）和 V4-Flash（284B / 13B 激活）发布预览版，1M 上下文，对标 Claude Opus 4.6 / GPT-5.4 / Gemini 3.1，华为 Ascend 超节点确认支持（DeepSeek API Docs、CNBC、Simon Willison）。
  • GPT-5.5 落地一周：4 月 23 日 OpenAI 发布 GPT-5.5（标准 / Thinking / Pro 三档），SWE-bench Pro 58.6%，Terminal-Bench 2.0 SOTA 82.7%，但 Claude Opus 4.7 仍以 64.3% 领跑 SWE-bench Pro（OpenAI、CNBC）。
  • 新一轮"DeepMind 系"创业潮：David Silver 创办的 Ineffable Intelligence 拿下 11 亿美元、51 亿美元估值的种子轮（Sequoia / Lightspeed / Nvidia / Google 跟投），主打"无人类数据的强化学习超学习者"——欧洲历史最大种子轮（CNBC、TechCrunch、Sequoia）。
  • 法律风险面：Musk v. Altman 4 月 27 日完成九人陪审团遴选，4 月 28 日 Oakland 联邦法院开庭。诉求包括请求最高 1340 亿美元"不当得利"返还、要求 Altman / Brockman 离任并把 OpenAI 还原为非营利结构（CNBC、GeekWire）。

Frontier Labs / Frontier Model Radar

  • OpenAI：GPT-5.5 上线一周已铺到 Plus / Pro / Business / Enterprise 与 Codex；同周宣布与 Microsoft 解除排他、可在所有云上提供产品。GPT-6 现指"下一代"，按官方表述目标在 2026 年内发布，节奏远短于 GPT-4 → GPT-5（OpenAI 公告）。
  • Anthropic：Mythos Preview 仍在 Project Glasswing 受控释出（约 40 家 AWS / Apple / Cisco / CrowdStrike / Google / Microsoft / NVIDIA 等参与，未公开发售），印度财政部要求银行业"针对 Mythos 类风险提前加固 IT 系统"（Anthropic Red、Foreign Policy、Business Standard）。Claude Opus 4.7 在 SWE-bench Pro 维持 64.3% 的最强 agentic coding 评分。
  • Google DeepMind：Gemini 3.1 Pro 加 Deep Research Max（Gemini Enterprise Agent Platform 重塑），同时通过对 Anthropic 最高 400 亿美元投资把 TPU 资源绑定到对手最强模型上；DeepMind 内部 600+ 员工请愿要求拒绝 Gemini 进入"机密军事运营"（Google Blog、Fortune）。
  • DeepSeek：V4 Preview 4 月 24 日开源（1M 上下文、华为 Ascend 适配），定价显著低于美西闭源旗舰。Simon Willison 总结：“almost on the frontier, a fraction of the price”，是开源端推动定价 / 上下文曲线下移的本周最大事件。
  • Qwen：Qwen3.6-35B-A3B（4/16）与 Qwen3.6-27B（4/22）已发布；本周未有新旗舰，主线仍是稳定性与编程友好度。
  • xAI / Mistral / Meta：本周窗口无新旗舰发布。Meta 在 1 月以来累计签下逾 6GW 核电采购，本周仍处算力侧 buildout 节奏。

unverified frontier signals：

  • 多家媒体复述的"GPT-5.5 = Spud；GPT-6 留作下一代"为社区追踪派的解读，OpenAI 未直接确认 GPT-6 发布日期。

关键人物与社区信号

  • Sam Altman / Greg Brockman（OpenAI）：4 月 27 日起进入 Musk v. Altman 庭审周；同日发布 Microsoft 协议改写 X 帖：“Microsoft remains our primary cloud partner, but we are now able to make our products and services available across all clouds.”（Tom’s Hardware）。
  • Andrej Karpathy：4 月延续"AI 第二大脑"叙事，从代码生成转向用 LLM 维护研究 wiki（约 100 篇 / 40 万字），与本周 agent 平台讨论形成对照（Medium 整理）。
  • Simon Willison：4 月 27 日发文系统性回顾"已死的 AGI 条款"演化史，是社区对 Microsoft-OpenAI 重组最完整的二级解读；4 月 24 日另发 DeepSeek V4 上手（simonwillison.net、simonwillison.net）。
  • Mira Murati（Thinking Machines）：4 月 22 日宣布与 Google Cloud 多十亿美元 GB300 协议，本周在 GTC 2026 开源面板上力挺"开放系统作为创新基础设施"立场（Implicator.ai、NVIDIA Blog）。
  • Aravind Srinivas（Perplexity）：在 GTC 开放面板把"Perplexity Computer"放到比开源模型更上层的 agent 框架里讨论，应对 Microsoft Copilot Agent Mode 默认开启带来的桌面端竞争。
  • Misha Laskin（Reflection AI）：4 月 23 日 CNBC Squawk on the Street 公开 250 亿美元估值新一轮、强调"中国是全球最强开源模型来源"；与 Ineffable Intelligence 共同代表"DeepMind 系新一代 RL 创业"主线。

people_signal_summary：本周人物层主线是"DeepMind 系出走 + 解绑型协议谈判"，社区端围绕 AGI 条款消亡、DeepSeek V4 开源、GPT-5.5 / Claude Opus 4.7 codename 比较展开。

Energy

  • 本周公用事业层重点是"AI 200GW 缺口 + 104GW 退役"双向夹击：IEA 数据显示 2025 年数据中心电力消耗增长 17%，预计 2026 年 AI 数据中心耗电量逼近 1,050 TWh，等同跻身全球第五大用电"国"。
  • 核电管线从 2024 年底 25GW 上升到约 45GW 条件性 SMR 协议；Meta 1 月起公布的多 GW 核电采购仍在堆叠（已逾 6GW），是本周最大的"模型外资源消耗"信号（Bloomberg、IEA）。
  • Anthropic-Google 协议中的 5GW 算力承诺意味着新增电力承担方从 hyperscaler 直接挪到云租户主体，能源 PPA 与模型公司开始一体化。

Chips

  • NVIDIA：4 月 27 日股价收报 216.61 美元、市值站上 5 万亿美元创新高；Rubin（Vera Rubin）平台首批系统将于 2026 下半年通过 AWS / Microsoft / Google 上线，对比 Blackwell 推理 token 成本降 10 倍、训练 GPU 数量降 4 倍（NVIDIA Newsroom、IBT）。
  • 华为 Ascend：与 DeepSeek V4 同日确认 Ascend 超节点支持 V4 推理，是中国侧"非英伟达堆栈"对外表态最完整的一周（CNBC）。
  • Intel-Google：4 月 18 日宣布多年期 AI 基建合作仍在底层支撑 Google 自研 + 第三方算力布局。
  • Anthropic 双供应链：通过 Amazon Trainium + Google TPU 同时获得两家自研芯片产能，是本周 chips 层最具结构性意义的"乙方对甲方反制"。

Infra

  • Microsoft × OpenAI 解绑：是过去两年 cloud-AI 绑定关系的最大结构性转折——OpenAI 模型可在 AWS / Google Cloud 部署、Microsoft 不再独家占有 IP；Microsoft 同时不必再向 OpenAI 支付反向营收分成（VentureBeat）。
  • Google Cloud Next 2026：力推 A2A（Agent-to-Agent）协议、Workspace Studio、Gemini Enterprise Agent Platform，配套 Anthropic 投资形成"自家模型 + 敌家模型 + 共享 TPU"的全栈三位一体（The Next Web）。
  • 算力地理：Stargate（OpenAI / Abilene 德州）+ Anthropic 多 GW TPU 区域 + Meta 多核电站 PPA + DeepSeek 华为 Ascend 集群——超大模型部署的电力 / 芯片 / 地理三联绑定开始制度化。

Model

  • GPT-5.5：SWE-bench Pro 58.6%、Terminal-Bench 2.0 82.7% SOTA、Intelligence Index 59；三档（Standard / Thinking / Pro）；4/24 起进入 API（Vellum 概览）。
  • Claude Opus 4.7：SWE-bench Pro 64.3% 仍领先；4 月 16 日 Anthropic 官宣（Anthropic 官方）。
  • Claude Mythos Preview：未对公众开放；本周印度财政部直接通报银行业警惕 Mythos 风险，是非美监管层第一次把单个模型作为系统性威胁正式预警。
  • DeepSeek V4 Pro / Flash：1M 上下文、$/Mtoken 显著低位，开源权重 + 华为兼容是中国侧 model 层"成本曲线"最强武器（Hugging Face）。
  • Gemini 3.1 Pro / Flash-Live + Deep Research Max：Google 模型层这周转向"长任务多模态 + agent 基础设施一体化"。
  • Ineffable Intelligence superlearner：David Silver 押注"无预训练、无人类数据"的 RL 路径，是 model 层的非主流押注，但能拉走 Sequoia / Lightspeed / Nvidia / Google 资本。

Application

  • Microsoft 365 Copilot Agent Mode 默认开启：4 月 22 日 GA，Word / Excel / PowerPoint 在缺省状态下进入 agentic 写作 / 分析 / 演示流程，是企业生产力侧最大变化（Asanify 概述）。
  • Anthropic 占据 40% 企业 LLM API 支出，OpenAI 从 2023 年的 50% 降至 27%（DesignRush / 行业统计）；与本周 OpenAI 解绑 Azure 行动直接相关——OpenAI 必须从企业端单点突破 Anthropic 的 API 主导。
  • C3 AI Code 发布：自然语言到生产级 AI 应用，“自动化整个开发生命周期”——是 application 层"代码生成 → 完整应用生成"的具体案例（行业引述）。
  • Perplexity Computer / Thinking Machines × Google Cloud：本周 application × infra 边界产品方向最清晰的两个名字。
  • Google 员工反 Pentagon Gemini 协议：超过 600 名 Google / DeepMind 员工请愿，反映 application 落地侧的内部治理摩擦正在升级。

层间联动影响

  • Energy ↔ Chips ↔ Infra：Anthropic = AWS Trainium + Google TPU + 5GW 算力绑定 = 一家模型公司同时穿透三大 hyperscaler 的电力 / 芯片 / 数据中心栈，是本周最强结构性"乙方反向锁定甲方"案例。
  • Infra ↔ Model：Microsoft-OpenAI 排他终结后，模型对云的分布式部署成为常态；OpenAI 对企业云方解绑、Anthropic 对超大资本绑定，二者代表两种相反的"生存策略"。
  • Model ↔ Application：Microsoft Copilot Agent Mode 把"模型 + 上下文 + agent 行为"打包进默认设置，Anthropic-Google 整合则把模型直接嵌入 Gemini Enterprise Agent Platform；GPT-5.5 + Codex + ChatGPT super app 仍处口风阶段。
  • People ↔ Capital ↔ Model：Ineffable Intelligence、Reflection AI、Periodic Labs、Humans& 等"DeepMind / OpenAI / Anthropic / xAI 离职 → 高估值新创"潮在本周再次集中爆发，会持续重塑后续 model 层路径选择。
  • 法律/治理 ↔ Frontier 架构：Musk v. Altman 庭审 + Mythos 监管 + Google 员工反军事使用 = 三条并行的非技术风险线，单独不致命，但叠加将进入未来 4-8 周的报告主线。

🎙️ 播客动态

  • 晚点聊 — 162: 批量生素材、模型筛网红，与飞书深诺 Meetsocial 沈晨岗聊 AI 时代的出海营销：飞书深诺 CEO 沈晨岗讨论 AI 在 2026 出海营销中的角色：从批量生成本地素材、模型化筛选网红，到中美 AI 营销生态差异。强调"廉价感 / 偷懒感"是 AI 创意广告最大风险，企业出海需重建审美与本地化能力。

候选池变化与后续关注

  • David Silver（Ineffable Intelligence 创始人 / CEO）：4 月 27 日完成 11 亿美元种子（51 亿美元估值），主张"超学习者无需人类数据"。Era of Experience 论文与 Sutton 合著，是 RL 后 LLM 时代最具结构主张的研究者之一。属 ai-generalists。
  • Misha Laskin（Reflection AI 联合创始人 / CEO）：4 月 23 日 CNBC Squawk on the Street 公开 250 亿美元估值，公开称"中国是全球最强开源模型源头"，Reflection 定位"美国侧开源前沿"。属 oss-ai-builders。
  • Ioannis Antonoglou（Reflection AI 联合创始人 / CTO）：AlphaGo / AlphaZero / MuZero 的研究工程师、Gemini RLHF 主导。本周与 Laskin 同步在媒体露出，是开源前沿 + RL 系一体化的代表声音。属 oss-ai-builders。

promoted_people：暂无（候选三人均为今日新增，需观察后续两次以上独立信号再考虑提级）。

观察点：

  • GPT-6 / Spud 后续节奏：Polymarket 老 4/14 窗口已落空（结果是 GPT-5.5），新 GPT-6 时点尚无强信号。
  • Mythos 受控释出后续：Project Glasswing 是否扩盘 / 监管反弹（特别是欧盟 / 印度银行业）。
  • Anthropic 资本化：650 亿美元（Google 400 + Amazon 250）兑现节奏与 5GW 落地里程碑。
  • DeepSeek V4 正式版 + 国内推理生态：1M 上下文价格曲线扩散对美西闭源旗舰的二级冲击。
  • Musk v. Altman 庭审：开庭周内的关键证词（特别是 Brockman / Altman 出庭）将成为未来 1-2 周 OpenAI 治理风险变量。
