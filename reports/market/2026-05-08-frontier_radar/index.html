Frontier Labs / Frontier Model Radar — 2026-05-08

5/7-5/8 frontier 主线由两大主轴驱动：(1) OpenAI 5/7 单日发布 6 条产品 / 安全 / 合作公告；(2) Anthropic 5/6-7 与 SpaceX/xAI Colossus 1 完成 300+ MW / 220 K+ GPU 算力交易，并探索轨道算力卫星。Google DeepMind 5/7 AlphaEvolve 长篇 impact 复盘把 AI for Math/Science 推到 Erdős/TSP/Ramsey 三项新纪录与 TPU 设计核心位置；Meta 5/8 公告 5/20 起 8 K 层级裁员；Mistral 一周前的 Vibe + Medium 3.5 落地；DeepSeek V4 折扣到期、Huawei Ascend 950 supernode 全栈对接。

OpenAI

  • GPT-5.5 Instant 默认化 + memory 升级（5/7）：ChatGPT Release Notes / Releasebot OpenAI — GPT-5.5 Pro 同步覆盖 Pro / Business / Enterprise，API 推出新一代语音智能模型。
  • GPT-5.5-Cyber 限量预览（5/7）：OpenAI 官博 / Trusted Access for Cyber / Help Net Security / Axios — Trusted Access 体系把 cyber 工作流权限释放给受验证团队。
  • Trusted Contact 自伤防护（5/7）：TechCrunch / BusinessToday — Adam Raine 案后 OpenAI 第二次大型 youth/safety 产品级动作。
  • ChatGPT 测试广告（5/7）：OpenAI 官博 / Distill 周报 — Free + Go 美区开始测试，Plus 以上无广告；Ads Manager beta 按 CPC 计价。
  • ChatGPT Futures: Class of 2026（5/7）：官博 / edu newsletter — 26 名学生 innovator，每人 $10K + frontier 模型访问。

Anthropic

  • SpaceX/xAI Colossus 1 300+ MW / 220K+ GPU 全量算力（5/6-7）：xAI 官博 / CNBC / DCD / Simon Willison / Al Jazeera — Pro/Max/Team/Enterprise Claude Code 5h 速率限额翻倍，Pro/Max 高峰封顶移除；与 SpaceX 联合研究 GW 级在轨 compute satellites。
  • Financial Services Agents（10 件）+ Microsoft 365 + Moody’s + FIS Financial Crimes Agent（5/5）：Anthropic 官博 / Bloomberg / Fortune / FIS PR — pitch builder / earnings reviewer / GL reconciler / KYC screener 等覆盖 banking / insurance / asset management 全岗位。
  • Mythos Project Glasswing 限量预览 + $900B 估值轮（持续）：Anthropic Mythos / CNBC $900B / Meyka FT 报道 — capital × procurement decoupling 主线对 5/1 Pentagon 黑名单事件形成对冲。

Google DeepMind

  • AlphaEvolve impact 复盘（5/7）：DeepMind 官博 / GitHub repository_of_problems / Decrypt / Pushmeet Kohli X — 已升级为 Google 基础设施核心组件并参与下一代 TPU 设计；Willow 量子分子模拟电路误差降 10x；与 Terence Tao / Javier Gómez-Serrano 合作刷新 Erdős / TSP / Ramsey 数下界；外溢到密码学、microeconomics、神经科学解释建模、合成数据、frontier AI safety mitigation。
  • CAISI Pre-release 政府评估协议（5/5）：CNBC / CNN / Claims Journal — 与 Microsoft / xAI 同时签订 pre-release 政府评估，与 OpenAI / Anthropic 并列。

Meta

  • 5/20 起 8K 裁员 + Superintelligence Labs Pods 重组（5/8 报道）：The Next Web / Meta about.fb / Meta AI blog / CNBC Wall Street 反馈 — 占员工 ~10%，下半年继续；Alexandr Wang 主导 MSL pods，2026 年 AI 基础设施 $115-135B；Muse Spark（4/8 闭源旗舰多模态）仍在产品化窗口。

xAI

  • Anthropic Colossus 1 算力出售 + Grok Imagine Quality Mode API + Connectors（5/6-7）：xAI 官博 / TechCrunch neocloud / mean.ceo Grok 月报 — xAI 自留 Colossus 2，把 Colossus 1 整体卖给 Anthropic；Grok Imagine 上 Quality Mode API、Web 端集成 Google Workspace / Outlook 等 Connectors。
  • Grok 4.1 Fast 等 deprecation 2 周通知 + 5/8 sentience-delusion 报道：TechSpot / Distill 周报 — 心理学家 Luke Nicholls 模拟测试 5 款模型，Grok 最易诱导妄想；开发者社区批评 deprecation 仓促。

Mistral
