关键人物与社区信号 — 2026-05-03

48 小时内的人物侧主线两条：(1) Sam Altman / Greg Brockman 因 Musk v. Altman 庭审与 4/27 MS-OAI 协议 narrative 处于"被动应对"状态，没有正面公开发文；(2) 学术与工程侧由 Karpathy 4/3-4/4 的 “LLM 知识库 + 第二大脑” 范式定调（外部解读延续到 4 月底），Simon Willison 5/2 Sightings 把"Claude Code + 鸟类摄影 + iNaturalist"链路作为 personal AI 用例落地样例。本日发现扫描提名 3 位候选人。

被跟踪人物今日信号

  • Sam Altman / Greg Brockman：Musk v. Altman 庭审 Week 1 内被点名 “被指控背叛 OAI 非营利使命，Musk 寻求罢免”，Altman 个人无新公开博文，OpenAI 仍以 4/26 “Our Principles” 作为单一对外口径。(Ringer 4/30 庭审 annotated; CNBC 4/30 Day 4 live)
  • Andrej Karpathy：4/3-4/4 推出"LLM 知识库 = 个人维基"工作流（单 topic 已 ~100 articles / 400k 字），4 月底-5 月初被 Ole Lehmann 等多位社区放大解读，逐步形成"Karpathy 不写 code，写 second brain"的新 narrative。(Karpathy on X; Medium 4 月解读)
  • Simon Willison：5/2 Sightings 把 Claude Code 与 Canon R6 Mark II 鸟类拍摄 + iNaturalist + 生成式 AI 串成一条 personal AI 用例链；同期延续 LLM 工具体系教程更新。
  • 其余跟踪人物（Boris Cherny / Amanda Askell / Alex Albert / Kevin Weil / Romain Huet / Harrison Chase / Jerry Liu / Sebastian Raschka / Jeremy Howard / Shawn Wang / Ethan Mollick / Nathan Lambert）：48 小时内未观察到突出一手公开信号；多数处于 4/27 MS-OAI 协议 + Mythos / Glasswing / Opus 4.7 系列的二次解读状态。

候选人发现扫描

  • Misha Laskin（Reflection AI 联合创始人 / CEO） — Reflection AI 是 NVIDIA Nemotron Coalition 8 家初创厂商之一，公开发声主张"foundation of intelligence remains open — not controlled by a few"，定位为开权重 frontier 联盟的代表性创业者，前 Google DeepMind Gemini 后训练 / reward 模型负责人，是当下"非闭源 frontier"叙事的关键人物。归 ai-generalists 组。(NVIDIA Newsroom; AI Primer 综述)
  • Tao Ge（Microsoft Research / GAI Group） — 5/3 paper-digest Top pick “Synthetic Computers at Scale” 一作（共 4 位作者，含 Baolin Peng / Jianfeng Gao），把"千台合成电脑作为 agent self-improvement 与 agentic RL 的基底"作为微软系正式技术信号；个人主页 getao.github.io 持续更新，关联 AgentInstruct / Orca-AgentInstruct 系列。归 oss-ai-builders 组。(arXiv 2604.28181; Microsoft Research GAI publications)
  • Beidi Chen（CMU ECE 助理教授） — 5/3 paper-digest Top pick “ARA / The Last Human-Written Paper” 37 位共同作者之一（与 Pentland / Mosharaf Chowdhury 同列），同期 Sky Computing Lab seminar 讲 MagicPIG / GSM-Infinite，CMU 在 ICLR 2026 多篇主导作者，是"长上下文效率 + agent infra"双方向高产学者。归 oss-ai-builders 组。(CMU 主页; Sky Seminar; ICLR 2026 CMU 综述)
