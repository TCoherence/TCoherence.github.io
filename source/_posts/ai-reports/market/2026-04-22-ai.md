# AI 日报｜2026-04-22

一句话结论：Anthropic Mythos 遭未授权访问事件升级引发安全治理争论，Amazon 追加 250 亿美元投资 Anthropic 重塑云-AI 格局，OpenAI Spud（GPT-5.5/6）发布窗口逼近，前沿模型竞争进入最密集阶段。

## 摘要

今日 AI 领域围绕三条主线展开：(1) Anthropic Claude Mythos Preview 被未授权第三方访问，TechCrunch/Bloomberg 同步报道，Sam Altman 公开批评 Anthropic 为"恐惧营销"，安全治理争论升温；(2) Amazon 宣布追加最高 250 亿美元投资 Anthropic（含 5GW 算力承诺），估值触及 3800 亿美元，年化收入突破 300 亿美元；(3) OpenAI Spud 预训练完成、Polymarket 给 4/30 发布 78% 概率，命名仍在 GPT-5.5 与 GPT-6 之间。五层方面，美国公用事业 1.4 万亿美元基建计划、NVIDIA Blackwell 售罄至年中、Intel-Google 深化合作均为基础设施层重大信号。

## Frontier Labs / Frontier Model Radar

**Anthropic — Mythos 安全事件升级**
- Claude Mythos Preview 自 4/7 发布以来持续引发关注。4/21 [TechCrunch 报道](https://techcrunch.com/2026/04/21/unauthorized-group-has-gained-access-to-anthropics-exclusive-cyber-tool-mythos-report-claims/)一个未授权群体通过猜测模型在线地址获取了 Mythos 访问权限，入口为第三方供应商环境。[Bloomberg 同步确认](https://www.bloomberg.com/news/articles/2026-04-21/anthropic-s-mythos-model-is-being-accessed-by-unauthorized-users)。
- Anthropic 声明称正在调查，"目前无证据表明核心系统受影响"。
- [NSA 据报也在使用 Mythos](https://techcrunch.com/2026/04/20/nsa-spies-are-reportedly-using-anthropics-mythos-despite-pentagon-feud/)，尽管与五角大楼存在分歧。
- [Sam Altman 公开回击](https://techcrunch.com/2026/04/21/sam-altman-throws-shade-at-anthropics-cyber-model-mythos-fear-based-marketing/)，称 Anthropic 策略为 "fear-based marketing"。
- **信号判断**：Mythos 是首个因"能力过强"而限制发布的前沿模型，其安全事件将重塑行业对受控发布模式的信任评估。

**OpenAI — Spud 发布倒计时**
- Spud（内部代号）预训练约 3/24 完成，[Polymarket 给 4/30 发布 78%](https://findskill.ai/blog/gpt-6-release-date/)，95%+ 6/30 前。
- 命名未定：性能跨代级则为 GPT-6，增量级则为 GPT-5.5。Sam Altman 称其为"very strong model"，Greg Brockman 称代表"两年研究成果"。
- GPT-5.4 已于 3/5 发布，首款原生 computer-use 通用模型。

**Google DeepMind — Gemini 3.1**
- Gemini 3.1 Pro 在语音和视觉多模态方面表现突出，此前报道 16 项基准中 13 项领先。

**中国实验室**
- [DeepSeek V4](https://evolink.ai/blog/deepseek-v4-release-window-prep/) "未来数周"发布，运行于华为芯片，多模态、100 万 token 上下文、Engram 记忆架构。美国正调查 DeepSeek 是否使用受限 NVIDIA Blackwell 芯片。
- [Qwen 3.6 Plus](https://renovateqr.com/blog/chinese-ai-models-april-2026) 4/2 发布，混合线性注意力 + 稀疏 MoE 架构，瞄准 agentic coding。

**未验证前沿信号**
- Mythos 未授权访问范围可能大于 Anthropic 目前确认——多个安全研究者暗示存在更多入口
- Spud 可能在 4/23-25 窗口发布（与 4/16 报告中 Polymarket 63% 预测一致，现已升至 78%）
- xAI Grok 4.3 beta 已推送给 SuperGrok Heavy 订阅用户，四代理交叉验证架构

## Energy

- **美国公用事业 1.4 万亿美元 AI 基建计划**：[PowerLines 分析](https://tech-insider.org/us-utility-1-4-trillion-ai-data-center-energy-2026/)显示 51 家公用事业公司规划至 2030 年投资 1.4 万亿美元，同比增长 27%。Duke Energy 1022 亿、Southern Company 812 亿领投。这是美国历史上最大规模的公用事业协调投资。
- **数据中心能耗加速**：美国数据中心 2023 年用电占比 4%，预计 2030 年达 9%。单机柜功率从 10-14kW 飙升至 100kW+，需要根本性的电力配送和冷却系统重设计。
- **英国议会调查低能耗计算**：[英国科技委员会开启调查](https://www.theregister.com/2026/04/20/mps_probe_low_energy_computing/)，探索低能耗计算能否遏制 AI 驱动的电力需求膨胀。

## Chips

- **NVIDIA Blackwell 售罄至年中**：每 GPU 约 4 万美元，超级计算客户（Microsoft, Meta, Amazon, Google）订单积压严重。[Blackwell Ultra 288 GPU 集群在 MLPerf Inference 创下系统级吞吐记录](https://nvidianews.nvidia.com/news/nvidia-blackwell-ultra-ai-factory-platform-paves-way-for-age-of-ai-reasoning)。
- **Intel-Google 深化合作**：[多年期合作协议](https://newsroom.intel.com/data-center/intel-google-deepen-collaboration-to-advance-ai-infrastructure)推进下一代 AI 与云基础设施，强调 CPU 和定制 IPU 在异构 AI 系统中的关键角色。
- **DeepSeek-华为芯片审查**：美国调查 DeepSeek 是否通过绕道方式获取受限 NVIDIA 芯片。DeepSeek V4 开发中使用华为 + 寒武纪芯片重写部分模型栈。

## Infra

- **Amazon-Anthropic 250 亿美元基建协议**：[Amazon 追加最高 250 亿美元投资](https://www.cnbc.com/2026/04/20/amazon-invest-up-to-25-billion-in-anthropic-part-of-ai-infrastructure.html)（初始 50 亿 + 里程碑绑定 200 亿），Anthropic 将在 AWS 上投入超 1000 亿美元/十年，获得高达 5GW 算力容量（含今年上线的 Trainium3）。Anthropic 估值 3800 亿美元，年化收入超 300 亿美元。
- **AAIF（Agentic AI Foundation）加速**：Linux Foundation 旗下 AAIF 成员已超 170 家，Block 捐赠 Goose、Anthropic 捐赠 MCP 规范，开源 agent 基础设施进入产业联盟阶段。
- **Anthropic 2027 年起使用多 GW 级下一代 TPU**：[Motley Fool 报道](https://www.fool.com/investing/2026/04/22/anthropic-just-announced-huge-news-for-alphabet-an/)对 Alphabet/Broadcom 产生利好。

## 层间联动影响

1. **能源→芯片→模型绑定深化**：1.4 万亿美元公用事业投资 + NVIDIA Blackwell 售罄 + 250 亿美元 Amazon-Anthropic 协议形成"算力-能源-模型"三角锁定。5GW 算力承诺意味着 Anthropic 的模型规模路线图直接受能源基础设施约束。
2. **Mythos 安全事件→应用层信任**：受控发布模型遭未授权访问，将迫使 Application 层重新评估前沿 API 的安全治理假设，特别是对 computer-use 和 cyber 能力的部署边界。
3. **Agent 框架标准化→模型层竞争转向**：AAIF 170+ 成员 + Microsoft Agent Framework GA + MCP v2.1 广泛采纳，模型竞争从裸基准转向"谁的 agent 生态最完整"。
4. **中国芯片自主→模型路线分化**：DeepSeek V4 使用华为/寒武纪芯片、美国出口管制调查同步推进，中美模型训练基础设施正式分叉。
