# Frontier Labs / Frontier Model Radar — 2026-05-04

5/1–5/4 的 frontier 主线被「美国国防部 IL6/IL7 七厂家协议」与 Anthropic「$30B ARR + $900B 估值谈判」两条线主导：OpenAI、Google、Microsoft、Amazon、Oracle、NVIDIA、SpaceX、Reflection AI 同日入选 Pentagon 分类网络供应商，Anthropic 因不让步军用安全护栏被显式排除（[CNN](https://www.cnn.com/2026/05/01/tech/pentagon-ai-anthropic) / [Tom's Hardware](https://www.tomshardware.com/tech-industry/artificial-intelligence/the-pentagon-announces-ai-deals-with-openai-google-microsoft-amazon-nvidia-and-more-llms-to-be-deployed-on-classified-department-of-war-networks-for-lawful-operational-use) / [Breaking Defense](https://breakingdefense.com/2026/05/pentagon-clears-7-tech-firms-to-deploy-their-ai-on-its-classified-networks/)）。Reflection AI 借 Misha Laskin 的 DeepMind 系背景拿到与七大同等的 IL6/IL7 资格，是本周 frontier watchlist 里第一次出现的「准 frontier」厂家。

OpenAI 同步推出 GPT-5.5-Cyber 与 Cybersecurity Action Plan，把 4/23 GPT-5.5 主线模型扩展到国防/网络防御方向（[Nextgov](https://www.nextgov.com/artificial-intelligence/2026/04/openai-makes-frontier-model-available-critical-cyber-defenders/413234/)）。Anthropic Mythos / Project Glasswing 同期被披露已扫出数千枚漏洞，但因军用安全条款分歧被 Pentagon 拉黑（[CNN](https://www.cnn.com/2026/05/01/tech/pentagon-ai-anthropic)）。Mistral 4/29 上架 Medium 3.5，是开源端唯一 5/1 周新增的 frontier 主线发布（[Codersera](https://codersera.com/blog/best-open-source-llm-2026-llama-4-qwen-3-5-deepseek-v4-gemma-4-mistral/)）。DeepSeek V4 / Qwen 3.5 / Gemma 4 主线本周无新版本，进入消化期。

## OpenAI

- **5/1–5/4，Pentagon IL6/IL7 入选 + GPT-5.5-Cyber 出货**：与 Google、Microsoft、Amazon、Oracle、NVIDIA、SpaceX、Reflection AI 同批进入分类网络，部署在 GenAi.mil 平台供 130 万国防人员调用；同期发布 GPT-5.5-Cyber 与 Cybersecurity Action Plan，把 4/23 GPT-5.5 旗舰扩展到国防安全场景。([CNN](https://www.cnn.com/2026/05/01/tech/pentagon-ai-anthropic) / [Nextgov](https://www.nextgov.com/artificial-intelligence/2026/04/openai-makes-frontier-model-available-critical-cyber-defenders/413234/) / [Breaking Defense](https://breakingdefense.com/2026/05/pentagon-clears-7-tech-firms-to-deploy-their-ai-on-its-classified-networks/))

## Anthropic

- **5/1–5/3，$30B ARR 公布同周被 Pentagon 排除**：Anthropic ARR 突破 $30B（5 个月内由 $9B 翻三倍），$1M+/年企业客户从 ~500 翻至 1000+；同时被 Pentagon 显式拉出 IL6/IL7 名单，理由是 Anthropic 拒绝放开军用通用条款（domestic mass surveillance / autonomous weapons 限制不变），Mythos / Project Glasswing 已经扫出数千枚漏洞但未改变国防部立场。([Yahoo Finance](https://finance.yahoo.com/news/anthropic-tops-30-billion-run-221045473.html) / [PYMNTS](https://www.pymnts.com/artificial-intelligence-2/2026/anthropic-hits-30-billion-run-rate-as-enterprise-demand-accelerates/) / [CNN](https://www.cnn.com/2026/05/01/tech/pentagon-ai-anthropic))
- **4/29，$900B 估值新一轮融资讨论曝光**：CNBC 披露 Anthropic 与投资人初谈下一轮估值至 $900B，超越 OpenAI 当前估值。([CNBC](https://www.cnbc.com/2026/04/29/anthropic-weighs-raising-funds-at-900b-valuation-topping-openai.html))

## Google DeepMind

- **5/1–5/4，Pentagon IL6/IL7 入选 + 4/22 Deep Research Max 已落地**：与 OpenAI / Microsoft / Amazon 同批拿到分类网络资格；Gemini 3.1 Pro 驱动的 Deep Research Max 4/22 上线，针对长程研究 workflow，含 MCP 支持与原生可视化。([CNN](https://www.cnn.com/2026/05/01/tech/pentagon-ai-anthropic) / [Google Blog](https://blog.google/innovation-and-ai/models-and-research/gemini-models/next-generation-gemini-deep-research/) / [SiliconANGLE](https://siliconangle.com/2026/04/22/google-launches-ai-research-agents-powered-gemini-3-1-pro/))

## Meta

- **本周无新 frontier 模型发布**：注意 Meta 此前已与 Vistra / TerraPower / Oklo / Constellation 签约获得 ~6.6 GW 核电承诺，PJM 区 1.2 GW 校园 2030 上线，是 frontier-lab 中能源侧最深的卡位。本周无主线模型新动作。([Meta](https://about.fb.com/news/2026/01/meta-nuclear-energy-projects-power-american-ai-leadership/))

## 未验证 frontier 信号

- **Reflection AI 是否拥有可对标 Pentagon IL6/IL7 的自有 frontier 模型**：Pentagon 把 Reflection AI 与 OpenAI / Google / Microsoft / Amazon / NVIDIA / SpaceX 同批纳入，但 Reflection AI 至今未公开发布商业 LLM；TechCrunch 此前报道目标为「数十万亿 token 训练的语言模型」，模型卡 / 基准未公开。该入选属 Pentagon 对「frontier 候选厂家」的押注，而非对已发布模型的认可。([TechCrunch](https://techcrunch.com/2026/05/01/pentagon-inks-deals-with-nvidia-microsoft-and-aws-to-deploy-ai-on-classified-networks/))
